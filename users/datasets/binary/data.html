"data=" defines the filename to the binary data. It is defined by DDS when creating the dataset -- either as a default filename or overridden by the user (see I/O Control). If the binary data is attached to a Dictionary, the value is "dict:". Some examples are: 

data=file_name

data=/data/project1/area_@

comment= binary data filename is "line33-"
data= line33-

comment= binary data is attached to Dictionary
data= dict:

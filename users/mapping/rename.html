The mapping Dictionary may also contain "rename:" Definitions which are used to rename Trace Headers going from one binary format to another. They take on the form: 

rename:in.field:out= name
where
  • "in" is the input format name (eg. "usp" or "segy").

  • "field" can be any Trace Header name in the input dataset.

  • "out" is the output format name (eg. "usp" or "segy").

  • "name" is the new Trace Header field name.

For example, segy header words "SrcX" can be reassigned to the new name, "SrPtXC" for the asp format. 

rename:segy.SrcX:asp= SrPtXC

When converting from another format to su, the Trace Headers get mapped over to the new format. Any Trace Header fields that exist in both formats with the same name ("Spare.Hdr.RecNum" is not the same as "RecNum"), automatically get mapped by default. All other default map Definitions for mapping to su are provided in "${DDSROOT}/src/lib/dds3/segy.map.dds" (Since su is a subset of segy format. Each map Definition name is prefixed by "map:*:su." (or the specific format can be substituted for the "*". (See Mapping for further detail).

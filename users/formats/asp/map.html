When converting from another format to asp, the Trace Headers get mapped over to the new format. Any Trace Header fields that exist in both formats with the same name ("Spare.Hdr.RecNum" is not the same as "RecNum"), automatically get mapped by default. All other default map Definitions for mapping to asp are provided in "${DDSROOT}/src/lib/dds3/asp.map.dds". Each map Definition name is prefixed by "map:*:asp." (or the specific format can be substituted for the "*". (See Mapping for further detail). 

asp mapping            segy expression
--------------------   ------------------------
map:segy:asp.SGRDat=   dateint(Date)
map:segy:asp.SGRNum=   MuteEnd
map:segy:asp.SGRAmp=   DateType
map:segy:asp.FlReFN=   Spare.Hdr.FlReFN
map:segy:asp.InStAp=   SrcStatic

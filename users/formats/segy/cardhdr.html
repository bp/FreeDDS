segy card header, SEGY_CARD, contains simple text and is available as a Dictionary Definition, segy_card3200=. There are up to 40 lines with up to 80 characters each. Each line is represented by a Definition Token in segy_card3200=, so it is important to be careful that the command line shell does not strip off the quotes ('"') in the definition. Better to put it into a parfile.

This may be examined in the Dictionary by end users or retrieved by application programs. End users may easily override this text with something like: 

comment= override card header value
segy_card3200=
    "C 1 CLIENT AMOCO                  COMPANY..."
    "C 2 LINE            AREA WEST DELTA #75/90..."
    "C 3 REEL NO 000014    DAY-START OF REEL..."
    ...
    "C40 END EBCDIC..."

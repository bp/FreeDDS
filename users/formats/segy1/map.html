When converting from another format to segy1, the Trace Headers get mapped over to the new format. Any Trace Header fields that exist in both formats with the same name ("Spare.Hdr.RecNum" is not the same as "RecNum"), automatically get mapped by default. All other default map Definitions for mapping to segy1 are provided in "${DDSROOT}/src/lib/dds3/segy1.map.dds". Each map Definition name is prefixed by "map:*:segy1." (or the specific format can be substituted for the "*". (See Mapping for further detail).

map:usp:segy1.MuteEnd=     SGRNum
map:usp:segy1.DateType=    SGRAmp
map:usp:segy1.SrcStatic=   InStAp
map:usp:segy1.GrpStatic=   RcStAp
map:usp:segy1.SrcX=        SrPtXC
map:usp:segy1.SrcY=        SrPtYC
map:usp:segy1.GrpX=        RcPtXC
map:usp:segy1.GrpY=        RcPtYC
map:usp:segy1.SrcDepth=    ShtDep
map:usp:segy1.UpholeTmSrc= UphlTm
map:usp:segy1.HorzStack=   FoldNm
map:usp:segy1.CdpTrcNum=   TrcNum

map:usp:segy1.GrpDatum=    RfSrEl
map:usp:segy1.TotalStatic= StaCor
map:usp:segy1.TrcIdCode=   (StaCor - 30000) ? 1 : 2

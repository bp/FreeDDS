Mnemonic Names for Internal Built-in Dictionarys:

There are several internal dictionaries built-in to DDS. These Dictionary names all end with the ":" (colon) character to distinguish them from ordinary dictionaries. (See Special Dictionaries for the special "format" and "mapping" dictionaries. 

Mnemonic     Dictionary Description
--------     ----------------------
stdin:       standard input
stdout:      standard output
par:         control parameters ("parfile" + "cmd:")
cmd:         command line arguments
env:         environment variables
hist:        history from open binary Dictionary
override:    override Definitions (for developers)
default:     default Definitions (for developers)
log:         error and diagnostic message logging
fdn:         file descriptor (fd7:, ...)
tmpxxx:      temporary (tmp4:, tmp_vel:, ...)

segy.fmt:    default segy format
segy1.fmt:   default segy1 format
usp.fmt:     default usp format
asp.fmt:     default asp format

segy.map:    default segy mapping
segy1.map:   default segy1 mapping
usp.map:     default usp mapping
asp.map:     default asp mapping

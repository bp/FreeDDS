fdds_info - control writing to a print file & console

SYNOPSIS

#include <fdds.h> (fixed-format sources)
#include <f90dds.h> (free-format sources)

integer function fdds_info(enable)

integer enable

DESCRIPTION

fdds_info is used to enable or disable printing data information when a file is opened up depending on whether enable is True (1) or False (0). Be default, printing of this information is enabled by fdds_openpr. This can be useful when opening and reopening the same file(s) repeatedly. The return value is always 0.

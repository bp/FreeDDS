fdds_align - fetch field alignment

SYNOPSIS

#include <fdds.h> (fixed-format sources)
#include <f90dds.h> (free-format sources)

integer fdds_align(bin_tag, field_tag)

integer bin_tag
integer field_tag

DESCRIPTION

fdds_align returns byte alignment (effective) associated with the field_tag. A value of -1 is returned on any error.

INPUT

bin_tag is a binary tag, previously obtained from fdds_open. It specifies a particular binary stream.

field_tag is the tag of the field for which the information is desired.

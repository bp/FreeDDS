fdds_memuse - query allocate memory

SYNOPSIS

#include <fdds.h> (fixed-format sources)
#include <f90dds.h> (free-format sources)

real*8 fdds_memuse( )

DESCRIPTION

fdds_memuse returns the amount of memory allocated through the DDS allocation routines. The DDS memory routines allow for runtime memory checking.

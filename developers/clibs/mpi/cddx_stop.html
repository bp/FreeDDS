cddx_stop - stop gracefully when using DDS and MPI

SYNOPSIS

#include <pcddx.h>

int cddx_stop(np, node)

int np
int node

DESCRIPTION

The function, "cddx_stop", sums all the errors reported by the different MPI threads through cdds_prterr.
It then stops the program gracefully, stopping all MPI threads.
Note that cddx_stop should be called by all MPI threads.

INPUT

The parameter, "np", specifies the number of MPI threads used by the program.

The parameter, "node", specifies the number of the current MPI thread.

RETURN VALUE

cddx_stop never returns.

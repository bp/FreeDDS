cddx_getdim - find the dimensions of a file.

SYNOPSIS

#include <pcddx.h>

int cddx_getdim(bin, rank, origin, delta, size, cumsize)

int bin
int *rank
float origin[RANK_MAX]
float delta[RANK_MAX]
int size[RANK_MAX]
long cumsize[RANK_MAX]

DESCRIPTION

cddx_getdim scans the dictionary to find the dimensions of a file.
If the file is regular (eg, not a pipe) then it also checks that actual number of traces matches the number indicated in the dictionary.
If they are different, then cumsize[0] will be -(the actual number of traces), else cumsize[0] will be 1.

INPUT

The parameter, "bin", specifies the binary tag of the file.

OUTPUT

The parameter, "rank", specifies the rank (number of dimensions) of the file.

The parameter, "origin", specifies the origin of each axis.

The parameter, "delta", specifies the delta spacing of each axis.

The parameter, "size", specifies the size of each axis.

The parameter, "cumsize", specifies the cumulative size in traces of each axis.
If the file is regular (eg, not a pipe) then it also checks that actual number of traces matches the number indicated in the dictionary.
If they are different, then cumsize[0] will be -(the actual number of traces), else cumsize[0] will be 1.

RETURN VALUE

cddx_getdim always returns 0.

cddx_jobrange - calculate the range of jobs to process by this MPI node

SYNOPSIS

#include <pcddx.h>

int cddx_jobrange(njobs, np, node, istart, iend)

int njobs
int np
int node
int *istart
int *iend

DESCRIPTION

The function, "cddx_jobrange", calculates the first and last "jobs" for this MPI node.
It also prints a message stating the node number and the job range.

INPUT

The parameter, "njobs", specifies the total number of jobs.

The parameter, "np", specifies the number of MPI threads used by the program.

The parameter, "node", specifies the number of the current MPI thread.

OUTPUT

The parameter, "istart", specifies the first job to process.

The parameter, "iend", specifies the last job to process.

RETURN VALUE

cddx_jobrange always returns 0.

cddx_trace_in_range - check if a trace is within the user specified range.

SYNOPSIS

#include <pcddx.h>

int cddx_trace_in_range(itrace, rank, cumsize, ss, ee, ii, dimstart, dimend)

int itrace
int rank
long cumsize[RANK_MAX]
int ss[RANK_MAX]
int ee[RANK_MAX]
int ii[RANK_MAX]
int dimstart
int dimend

DESCRIPTION

The function, "cddx_trace_in_range", checks if a trace is within the range specified by the user using the 2s= 2e= 2i= parameters.
cddx_trace_in_range is thread safe and so can be called within an OpenMP loop.

INPUT

The parameter, "itrace", specifies the trace number (the first trace is numbered 0).

The parameter, "rank", specifies the rank (number of dimensions) of the file.

The parameter, "cumsize", specifies the cumulative size, in traces, of each axis.

The parameter, "ss", specifies the user specified starting value to process for each axis.

The parameter, "ee", specifies the user specified end value to process for each axis.

The parameter, "ii", specifies the user specified increment to process for each axis.

The parameter, "dimstart", specifies the first dimension to check. For example, if you only want to know if the trace is within the user specified range for the 3rd axis and above, set this to 3. Valid entries are from 2 to rank.

The parameter, "dimend", specifies the last dimensions to check. Valid entries are from 2 to rank.

RETURN VALUE

cddx_trace_in_range returns 1 if the trace is within the user range, 0 otherwise.

cddx_RK_show - write summary details of a Runge-Kutta extrapolator to the print file.

SYNOPSIS

#include <cddx_rungekutta.h>

int cddx_RK_show(rk_tag)

int rk_tag

DESCRIPTION

cddx_RK_show writes summary details of a Runge-Kutta extrapolator to the print file.

INPUT

The parameter, "rk_tag", specifies which Runge-Kutta extrapolator to show. Use function cddx_RK_new to create the extrapolator.

RETURN VALUE

cddx_RK_show always returns 0.

cddx_RK_new - create a new Runge-Kutta extrapolator.

SYNOPSIS

#include <cddx_rungekutta.h>

int cddx_RK_new(alias, nn, RHS)

const char *alias
int nn
int (*RHS)(int *tag, int *nn, float *vec_in, float *vec_out )

DESCRIPTION

cddx_RK_new creates a new Runge-Kutta extrapolator.
It scans for user input to decide which type of Runge-Kutta method to use.
Currently implemented methods are 2nd (rk2) and 4th order (rk4).
The user can specify which method to use by specifying rk=rk2 or rk=rk4.

INPUT

The parameter, "alias", specifies a prefix to add to the rk= keyword to allow the use of multiple different Runge-Kutta methods with the same application.

The parameter, "nn", specifies the maximum size of the vector to propagate.

The parameter, "RHS", specifies a function to compute the Right Hand Size of the equations.
The parameters for RHS are:

  • tag: pointer for internal data needed by RHS.
  • nn: the size of the vector to propagate. Note that nn is a pointer for compatibility with Fortran.
  • vec_in: input data vector.
  • vec_out: output data vector.

RETURN VALUE

cddx_RK_new returns a tag to the newly created Runge-Kutta Extrapolator. In the event of an error, it will return -1.

cddx_RK_step - take a step using a Runge-Kutta extrapolator.

SYNOPSIS

#include <cddx_rungekutta.h>

int cddx_RK_step(rk_tag, dt, yin, dydt, yout, dyout)

int rk_tag
float dt
float *yin
float *dydt
float *yout
float *dyout

DESCRIPTION

cddx_RK_step takes a step using Runge-Kutta extrapolator.
NOTE that this function is not re-entrant, which means that you cannot use it inside an OpenMP parallelized loop.
Each thread must use a separate extrapolator, created using cddx_RK_new.

INPUT

The parameter, "rk_tag", specifies which Runge-Kutta extrapolator to show. Use function cddx_RK_new to create the extrapolator.

The parameter, "dt", specifies the step length.

The parameter, "yin", specifies the input position vector.

The parameter, "dydt", specifies the input direction vector.

The parameter, "yout", specifies the input position vector.

The parameter, "dyout", specifies the input direction vector.

RETURN VALUE

cddx_RK_step returns 0 for normal termination, or -1 if there was an error.

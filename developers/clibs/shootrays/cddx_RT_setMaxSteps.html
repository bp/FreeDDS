cddx_RT_setMaxSteps - set the maximum number of steps that the raytracer should take.

SYNOPSIS

#include <cddx_raytracer.h>

int cddx_RT_setMaxSteps(rt_tag, nsteps)

int rt_tag
int nsteps

DESCRIPTION

cddx_RT_setMaxSteps sets the maximum number of steps that the raytracer should take.

INPUT

The parameter, "rt_tag", specifies for which raytracer to return the maximum number of steps.

The parameter, "nsteps", specifies the maximum number of steps.

RETURN VALUE

cddx_RT_setMaxSteps returns maximum number of steps that the raytracer should take.

cddx_RT_numSteps - returns the number steps of the ray.

SYNOPSIS

#include <cddx_raytracer.h>

int cddx_RT_numSteps(rt_tag)

int rt_tag

DESCRIPTION

cddx_RT_numSteps returns the number of steps of the ray.

INPUT

The parameter, "rt_tag", specifies for which raytracer to return the number of steps.

RETURN VALUE

cddx_RT_numSteps returns the number of steps of the ray.

cddx_RT_maxTime - returns the maximum traveltime of ray before the raytracer stops.

SYNOPSIS

#include <cddx_raytracer.h>

float cddx_RT_maxTime(rt_tag)

int rt_tag

DESCRIPTION

cddx_RT_maxTime returns the maximum traveltime of ray before the raytracer stops.

INPUT

The parameter, "rt_tag", specifies for which raytracer to return the maximum traveltime.

RETURN VALUE

cddx_RT_maxTime returns the maximum traveltime of ray before the raytracer stops.

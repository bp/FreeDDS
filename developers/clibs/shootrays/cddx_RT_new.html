cddx_RT_new - create a new raytracer.

SYNOPSIS

#include <cddx_raytracer.h>

int cddx_RT_new(alias, vel_tag, rk_tag)

const char *alias
int vel_tag
int rk_tag

DESCRIPTION

cddx_RT_new creates a new raytracer.
It scans for user input to decide which type of raytracing method to use.
Currently only isotropic is implemented.
Eventually, the user will be able to specify which method to use by specifying raytracer=isotropic or raytracer=tti, for example.

The raytracer will ensure at run-time that it has all of the parameters for it to run successfully.
(ie. TTI raytracer will check that the user provides, epsilon, delta, tilts, etc.)

INPUT

The parameter, "alias", specifies a prefix to add to the raytracer= keyword to allow the use of multiple different raytrace methods with the same application.

The parameter, "vel_tag", specifies a tag for the velocity to use.

The parameter, "rk_tag", specifies a tag for the Runge-Kutta method to use.

RETURN VALUE

cddx_RT_new returns a tag to the newly created raytracer. In the event of an error, it will return -1.

cddx_RT_setMaxTime - set the maximum traveltime of ray before the raytracer stops.

SYNOPSIS

#include <cddx_raytracer.h>

float cddx_RT_setMaxTime(rt_tag, maxtime)

int rt_tag
float maxtime

DESCRIPTION

cddx_RT_setMaxTime sets the maximum traveltime of ray before the raytracer stops.

INPUT

The parameter, "rt_tag", specifies for which raytracer to return the maximum traveltime.

The parameter, "maxtime", specifies the maximum traveltime.

RETURN VALUE

cddx_RT_setMaxTime returns the maximum traveltime of ray before the raytracer stops.

int cddx_RT_updateOutput( int rt_tag, int step, float vec_x[6], float vec_rhs[6], float dt, float dtsm, float *output );

cddx_RT_updateOutput - update the output array for current point on the ray.

SYNOPSIS

#include <cddx_raytracer.h>

int cddx_RT_updateOutput(rt_tag, step, vec_x, vec_rhs, dt, dtsm, output)

int rt_tag
float step
float vec_x[6]
float vec_rhs[6]
float dt
float dtsm
float *output

DESCRIPTION

cddx_RT_updateOutput updates the output array for current point on the ray.
Internally, the raytracer will calculate a number of attributes along the ray, many of which you may not need.
Use the function cddx_RT_requestOut to specify which parameters you need.
cddx_RT_updateOutput is then called to populate the output array.

INPUT

The parameter, "rt_tag", specifies which raytracer to show.
Use function cddx_RT_new to create the raytracer.

The parameter, "step", specifies the ray step length.

The parameter, "vec_x", specifies the ray point location and slowness vector.

The parameter, "vec_rhs", specifies the right hand side of the ray equations for this ray point.

The parameter, "dt", specifies the traveltime change from the previous point on the ray,
where the traveltime is calculated in the non-smoothed velocity model.

The parameter, "dtsm", specifies the traveltime change from the previous point on the ray,
where the traveltime is calculated in the smoothed velocity model.

OUTPUT

The parameter, "output", specifies the array for the output parameters.
Note that output should be of sufficient size, which will be the number of times cddx_requestOut has been successfully called (ie the number of parameters requested), times the max. number of raysteps (see also cddx_RT_maxSteps).

RETURN VALUE

cddx_RT_updateOutput always returns 0, unless there is an error, in which case it returns -1.

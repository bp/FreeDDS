cddx_RT_requestOut - request an output parameter from the raytracer.

SYNOPSIS

#include <cddx_raytracer.h>

int cddx_RT_requestOut(rt_tag, param)

int rt_tag
ENUM param

DESCRIPTION

cddx_RT_requestOut requests an output parameter from the raytracer.
This function should be called before any raytracing takes place.

INPUT

The parameter, "rt_tag", specifies for which raytracer to request the output parameter.

The parameter, "param", specifies the output parameter.
Currently supported values are:
VALUE DESCRIPTION
cddx_RT_X X location
cddx_RT_Y Y location
cddx_RT_Z Y location
cddx_RT_PX X-component of the slowness vector
cddx_RT_PY Y-component of the slowness vector
cddx_RT_PZ Z-component of the slowness vector
cddx_RT_UX X-component of the group velocity vector
cddx_RT_UY Y-component of the group velocity vector
cddx_RT_UZ Z-component of the group velocity vector
cddx_RT_TX X-component of the tangent to the ray
cddx_RT_TY Y-component of the tangent to the ray
cddx_RT_TZ Z-component of the tangent to the ray
cddx_RT_LENGTH Ray length
cddx_RT_TIME Traveltime
cddx_RT_SMOOTHTIME Traveltime calculated in a smooth velocity
cddx_RT_VPH Magnitude of the phase velocity in the slowness direction
cddx_RT_VGP Magnitude of the group velocity in the ray direction
cddx_RT_VRMS RMS velocity
cddx_RT_DVDX X-derivative of the velocity
cddx_RT_DVDY Y-derivative of the velocity
cddx_RT_DVDZ Z-derivative of the velocity

RETURN VALUE

cddx_RT_requestOut returns an index into the output array of param.
If the raytracer does not support param then cddx_RT_requestOut returns -1.

cddx_RT_error - returns the raytracer error status.

SYNOPSIS

#include <cddx_raytracer.h>

int cddx_RT_error(rt_tag)

int rt_tag

DESCRIPTION

cddx_RT_error returns the raytracer error status.

INPUT

The parameter, "rt_tag", specifies for which raytracer to return the error stauts.

RETURN VALUE

cddx_RT_error always returns 0.

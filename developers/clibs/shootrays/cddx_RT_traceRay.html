cddx_RT_traceRay - trace a ray.

SYNOPSIS

#include <cddx_raytracer.h>

int cddx_RT_traceRay(rt_tag, xs, ys, zs, decl, azm, output)

int rt_tag
float xs, ys, zs
float decl
float azm
float *output

DESCRIPTION

cddx_RT_traceRay traces a ray.

INPUT

The parameter, "rt_tag", specifies which raytracer to show.
Use function cddx_RT_new to create the raytracer.

The parameter, "xs, ys, zs", specifies the source location.

The parameter, "decl", specifies the ray declination.

The parameter, "azm", specifies the ray azimuth.

OUTPUT

The parameter, "output", specifies the array for the output parameters.
Note that output should be of sufficient size.
See the function cddx_requestOut for defining which parameters to output.

RETURN VALUE

cddx_RT_traceRay always returns 0, unless there is an error, in which case it returns -1.

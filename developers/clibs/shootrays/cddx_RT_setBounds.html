cddx_RT_setBounds - set the bounds within which to trace rays.

SYNOPSIS

#include <cddx_raytracer.h>

int cddx_RT_setBounds(rt_tag, xmin, xmax, ymin, ymax, zmin, zmax)

int rt_tag
float xmin
float xmax
float ymin
float ymax
float zmin
float zmax

DESCRIPTION

cddx_RT_setBounds checks if a point is within the raytracer bounds.

INPUT

The parameter, "rt_tag", specifies against which raytracer to check the bounds.

The parameter, "xmin", specifies the min x location to trace rays.

The parameter, "xmax", specifies the max x location to trace rays.

The parameter, "ymin", specifies the min y location to trace rays.

The parameter, "ymax", specifies the max y location to trace rays.

The parameter, "zmin", specifies the min z location to trace rays.

The parameter, "zmax", specifies the max z location to trace rays.

RETURN VALUE

cddx_RT_setBounds always returns 0.

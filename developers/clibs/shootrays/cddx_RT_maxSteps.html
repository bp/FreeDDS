cddx_RT_maxSteps - returns the maximum number of steps that the raytracer should take.

SYNOPSIS

#include <cddx_raytracer.h>

int cddx_RT_maxSteps(rt_tag)

int rt_tag

DESCRIPTION

cddx_RT_maxSteps returns the maximum number of steps that the raytracer should take.

INPUT

The parameter, "rt_tag", specifies for which raytracer to return the maximum number of steps.

RETURN VALUE

cddx_RT_maxSteps returns maximum number of steps that the raytracer should take.

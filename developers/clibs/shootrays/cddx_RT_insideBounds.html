cddx_RT_insideBounds - check if a point is within bounds.

SYNOPSIS

#include <cddx_raytracer.h>

int cddx_RT_insideBounds(rt_tag, point)

int rt_tag
float point(3)

DESCRIPTION

cddx_RT_insideBounds checks if a point is within the raytracer bounds.

INPUT

The parameter, "rt_tag", specifies against which raytracer to check the bounds.

The parameter, "point", specifies an input point.

RETURN VALUE

cddx_RT_insideBounds always returns 1 if the point is within the bounds, 0 otherwise.

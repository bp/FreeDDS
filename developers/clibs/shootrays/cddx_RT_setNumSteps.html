cddx_RT_setNumSteps - set the number steps of the ray.

SYNOPSIS

#include <cddx_raytracer.h>

int cddx_RT_setNumSteps(rt_tag, numsteps)

int rt_tag
int numsteps

DESCRIPTION

cddx_RT_setNumSteps sets the number of steps of the ray.

INPUT

The parameter, "rt_tag", specifies for which raytracer to return the number of steps.

The parameter, "numsteps", specifies the number of steps.

RETURN VALUE

cddx_RT_setNumSteps returns the number of steps of the ray.

cddx_RT_show - write summary details of a raytracer to the print file.

SYNOPSIS

#include <cddx_raytracer.h>

int cddx_RT_show(rt_tag)

int rt_tag

DESCRIPTION

cddx_RT_show writes summary details of a raytracer to the print file.

INPUT

The parameter, "rt_tag", specifies which raytracer to show. Use function cddx_RT_new to create the raytracer.

RETURN VALUE

cddx_RT_show always returns 0.

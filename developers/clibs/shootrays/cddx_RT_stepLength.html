cddx_RT_stepLength - returns the step length that the raytracer should take.

SYNOPSIS

#include <cddx_raytracer.h>

float cddx_RT_stepLength(rt_tag)

int rt_tag

DESCRIPTION

cddx_RT_stepLength returns step length that the raytracer should take.

INPUT

The parameter, "rt_tag", specifies for which raytracer to return the step length.

RETURN VALUE

cddx_RT_stepLength returns the step length that the raytracer should take.

cddz_hdtype - Detect the host-dependent type of a field

SYNOPSIS

#include "cdds.h"
#include "cddz.h"

const char *cddz_hdtype(bin_tag, field_tag)

BIN_TAG bin_tag
SYM_TAG field_tag

DESCRIPTION

Function "cddz_hdtype" returns the host-dependent type for a field. Host-dependent type is uniquely determined by the combination of "genus" and "precision". The table below shows the association between genus, precision, and host-dependent type. 
           bytes of
genus     precision    host_type

integer       1        signed char
integer       2        short
integer       4        int
integer       8        long long

float         4        float
float         8        double
float        16        long double

complex       8        complex
complex      16        complex double
complex      32        complex long
(Several genera have been omitted becuase they are seldom used: character, pointer, unsigned, structure, union, void)

INPUT

bin_tag is a binary tag, previously obtained from cdds_open. It specifies a particular binary stream. field_tag is the tag of the field for which the host-dependent type is desired. It can be found with "cdds_member".

EXAMPLE

int bin, tag;
const char *RecNum_type, *Sample_type;
 ...
/* host_type of RecNum field */
tag = cdds_member(bin, 0, "RecNum")
RecNum_type = cddz_hdtype(bin, tag);
 ...
/* host_type of Samples field */
tag = cdds_member(bin, 0, "Samples");
Sample_type = cddz_hdtype(bin, tag);

cddz_md5sum - Get the md5sum of a buffer

SYNOPSIS

#include "cddz.h"

const char *fddz_md5sum(buffer, byte_count)

void *buffer
long long byte_count

DESCRIPTION

Function "cddz_md5sum" gives programs access to GNU core utils function md5_buffer. This is the message-digest function used by utility md5sum(1). Access to the MD5 algorithm is useful when trying to locate system bugs that are causing headaches.

INPUT

buffer is the memory block to digest. byte_count tells "fddz_md5sum" the number of bytes to process.

SEE ALSO

   md5sum(1) compute and check MD5 message digest

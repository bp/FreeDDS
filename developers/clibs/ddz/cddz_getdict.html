cddz_getdict - Get the name of a dictionary when only the BIN_TAG is known

SYNOPSIS

#include "cdds.h"
#include "cddz.h"

const char *cddz_getdict(bin_tag)

int bin_tag

DESCRIPTION

Function "cddz_getdict" finds the name of a dictionary given it's BIN_TAG. This is useful when the name is needed after a file was openned with cddx_in or cddx_out functions.

EXAMPLE

int        ier;
BIN_TAG    fd_in, fd_out, fd_buf_out;
const char *buf_in_dict, *buf_out_dict, *Title;
 ...
/*
 * Use convenience routines on input but not on output
 */
fd_in = cddx_in ("in","stdin:",Title);
buf_in_dict = cddz_getdict(fd_in);
buf_out_dict = cdds_out(" "," ",buf_in_dict);
fd_buf_out = cdds_open(buf_out_dict," "," ","m");
out_dict = cdds_out(" "," ",buf_out_dict);
fd_out = cdds_open(out_dict," "," ", "w");

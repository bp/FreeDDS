cddz_scann - Get the name of a definition found by fdds_scan*()

SYNOPSIS

#include "cdds.h"
#include "cddz.h"

int cddz_scann(alias,defnname)

const char *alias
const char **defnname

DESCRIPTION

Function "cddz_scann" finds the "name" of the definition that is found by any of the cdds_scan*() functions. This is used to give applications a way around the "single alias name" restriction of convenience routines cddx_in*, cddx_mod*, or cddx_out*.

OUTPUT

When a definition is found, defnname is set to the "name" part of the definition and the length of the the name is returned.

When no definition is found, or if no dictionaries are in scan mode, defnname is set to NULL and EOF is returned.

EXAMPLE

int        ier;
const char *defnname, *Title;
BIN_TAG    fd_in;
 ...
/*
 * Use convenience routines to open input with more than one alias
 */
ier = cddz_scann("Vp Vz vel",defnname)
fd_in = cddx_in(defnname,"stdin:",Title);

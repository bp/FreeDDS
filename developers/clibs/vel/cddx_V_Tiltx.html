cddx_V_Tiltx - return a tag to a Tiltx parameter.

SYNOPSIS

#include <cddx_vel.h>

int cddx_V_Tiltx(vel_tag)

int vel_tag

DESCRIPTION

cddx_V_Tiltx returns a tag to the Tiltx parameter of a velocity.
If the velocity does not have Tiltx defined, then cddx_V_Tiltx returns -1.

INPUT

The parameter, "vel_tag", specifies a velocity for which to find the Tiltx parameter.

RETURN VALUE

cddx_V_Tiltx returns a tag to the Tiltx parameter.
If Tiltx is not defined, then it will return -1.

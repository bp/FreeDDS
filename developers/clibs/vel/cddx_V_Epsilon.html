cddx_V_Epsilon - return a tag to a Epsilon parameter.

SYNOPSIS

#include <cddx_vel.h>

int cddx_V_Epsilon(vel_tag)

int vel_tag

DESCRIPTION

cddx_V_Epsilon returns a tag to the Epsilon parameter of a velocity.
If the velocity does not have Epsilon defined, then cddx_V_Epsilon returns -1.

INPUT

The parameter, "vel_tag", specifies a velocity for which to find the Epsilon parameter.

RETURN VALUE

cddx_V_Epsilon returns a tag to the Epsilon parameter.
If Epsilon is not defined, then it will return -1.

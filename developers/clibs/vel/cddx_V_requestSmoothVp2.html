cddx_V_requestSmoothVp2 - read a SmoothVp2 parameter.

SYNOPSIS

#include <cddx_vel.h>

int cddx_V_requestSmoothVp2(vel_tag, critical)

int vel_tag
int critical

DESCRIPTION

cddx_V_requestSmoothVp2 reads a SmoothVp2 parameter.

INPUT

The parameter, "vel_tag", specifies a velocity with which to associate this SmoothVp2 parameter.
The keyword prefix for the users is "smoothVp2". See also function fddx_V_newElement.

The parameter, "critical", specifies whether this parameter is critical.
If "critical" is non-zero and cddx_V_requestSmoothVp2 is unable to find a suitable SmoothVp2 parameter, then a DDS error is printed.
NOTE that in the event of an error the program continues in order to allow for clean stopping of MPI codes.

RETURN VALUE

cddx_V_requestSmoothVp2 returns a tag to the newly created SmoothVp2 parameter.
If it is unable to create SmoothVp2, then it will return -1.

cddx_V_requestSmoothVs - read a SmoothVs parameter.

SYNOPSIS

#include <cddx_vel.h>

int cddx_V_requestSmoothVs(vel_tag, critical)

int vel_tag
int critical

DESCRIPTION

cddx_V_requestSmoothVs reads a SmoothVs parameter.

INPUT

The parameter, "vel_tag", specifies a velocity with which to associate this SmoothVs parameter.
The keyword prefix for the users is "smoothVs". See also function fddx_V_newElement.

The parameter, "critical", specifies whether this parameter is critical.
If "critical" is non-zero and cddx_V_requestSmoothVs is unable to find a suitable SmoothVs parameter, then a DDS error is printed.
NOTE that in the event of an error the program continues in order to allow for clean stopping of MPI codes.

RETURN VALUE

cddx_V_requestSmoothVs returns a tag to the newly created SmoothVs parameter.
If it is unable to create SmoothVs, then it will return -1.

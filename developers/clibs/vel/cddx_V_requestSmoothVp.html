cddx_V_requestSmoothVp - read a SmoothVp parameter.

SYNOPSIS

#include <cddx_vel.h>

int cddx_V_requestSmoothVp(vel_tag, critical)

int vel_tag
int critical

DESCRIPTION

cddx_V_requestSmoothVp reads a SmoothVp parameter.

INPUT

The parameter, "vel_tag", specifies a velocity with which to associate this SmoothVp parameter.
The keyword prefix for the users is "smoothVp". See also function fddx_V_newElement.

The parameter, "critical", specifies whether this parameter is critical.
If "critical" is non-zero and cddx_V_requestSmoothVp is unable to find a suitable SmoothVp parameter, then a DDS error is printed.
NOTE that in the event of an error the program continues in order to allow for clean stopping of MPI codes.

RETURN VALUE

cddx_V_requestSmoothVp returns a tag to the newly created SmoothVp parameter.
If it is unable to create SmoothVp, then it will return -1.

cddx_V_requestEpsilon - read a Epsilon parameter.

SYNOPSIS

#include <cddx_vel.h>

int cddx_V_requestEpsilon(vel_tag, critical)

int vel_tag
int critical

DESCRIPTION

cddx_V_requestEpsilon reads a Epsilon parameter.

INPUT

The parameter, "vel_tag", specifies a velocity with which to associate this Epsilon parameter.
The keyword prefix for the users is "epsilon". See also function fddx_V_newElement.

The parameter, "critical", specifies whether this parameter is critical.
If "critical" is non-zero and cddx_V_requestEpsilon is unable to find a suitable Epsilon parameter, then a DDS error is printed.
NOTE that in the event of an error the program continues in order to allow for clean stopping of MPI codes.

RETURN VALUE

cddx_V_requestEpsilon returns a tag to the newly created Epsilon parameter.
If it is unable to create Epsilon, then it will return -1.

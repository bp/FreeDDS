cddx_V_Vp2 - return a tag to a Vp squared parameter.

SYNOPSIS

#include <cddx_vel.h>

int cddx_V_Vp2(vel_tag)

int vel_tag

DESCRIPTION

cddx_V_Vp2 returns a tag to the Vp squared parameter of a velocity.
If the velocity does not have Vp2 defined, then cddx_V_Vp2 returns -1.

INPUT

The parameter, "vel_tag", specifies a velocity for which to find the Vp2 parameter.

RETURN VALUE

cddx_V_Vp2 returns a tag to the Vp2 parameter.
If Vp2 is not defined, then it will return -1.

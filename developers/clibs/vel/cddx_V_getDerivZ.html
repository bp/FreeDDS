cddx_V_getDerivZ - return the Z-derivative of a parameter at point (xx,yy,zz).

SYNOPSIS

#include <cddx_vel.h>

float cddx_V_getDerivZ(par_tag, xx, yy, zz)

int par_tag
float xx
float yy
float zz

DESCRIPTION

cddx_V_getDerivZ returns the Z-derivative of a parameter at point (xx,yy,zz).

INPUT

The parameter, "par_tag", specifies a tag to the parameter.

The parameters, "xx, yy, zz", specifies a location at which to evaluate the parameter.

RETURN VALUE

cddx_V_getDerivZ returns the Z-derivative of the parameter par_tag at point (xx,yy,zz).
NOTE: no error checking is done to ensure that par_tag is valid.

cddx_V_new - create a new velocity model.

SYNOPSIS

#include <cddx_vel.h>

int cddx_V_new(alias)

const char *alias

DESCRIPTION

cddx_V_new creates a new velocity model structure.
The structure itself will be empty. Use the cddx_V_request* functions to populate it with real data.

INPUT

The parameter, "alias", specifies a prefix to add to the keywords (see the cddx_V_request* functions)
to allow the use of multiple different velocities with the same application.
For example, if you want the user to be able to specify a P-wave velocity with the keyword old_vp=, then "alias" would be old_.

RETURN VALUE

cddx_V_new returns a tag to the newly created velocity model. In the event of an error, it will return -1.

NAME
       LSAME - Test if characters are equal, ignoring case.

SYNOPSIS
       LOGICAL FUNCTION LSAME( CA, CB )

           CHARACTER    CA, CB

DESCRIPTION
       LSAME returns .TRUE. if CA is the same letter as CB, ignoring case.


ARGUMENTS
       CA      (input) CHARACTER*1
       CB      (input) CHARACTER*1

       CA and CB specify the single characters to be compared.

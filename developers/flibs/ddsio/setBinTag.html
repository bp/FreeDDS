setBinTag - Sets this data object's binary tag

SYNOPSIS

use DdsIO_mod

procedure setBinTag(tag)

integer, intent(in) :: tag

DESCRIPTION

"setBinTag" is a type-bound procedure on a data object of type(DdsObject). This procedure associates an input binary tag with the data object.

EXAMPLE

	use DdsIO_mod, only : DdsObject
	
	type(DdsObject) :: inData
	
	call inData%setBinTag(-1)

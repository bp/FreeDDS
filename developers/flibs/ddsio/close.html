close - Closes data object

SYNOPSIS

use DdsIO_mod

procedure close()

DESCRIPTION

"close" is a type-bound procedure on a data object of type(DdsObject). This procedure closes the data object for further use.

EXAMPLE

	use DdsIO_mod, only : DdsObject
	
	type(DdsObject) :: inData
	
	! open data in read mode
	call inData%open('r','in')
	! close data
	call inData%close()

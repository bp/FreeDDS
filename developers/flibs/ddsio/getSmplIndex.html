getSmplIndex - Returns the index of the first sample in this data object's file

SYNOPSIS

use DdsIO_mod

integer(kind=DPI) function getSmplIndex()

DESCRIPTION

"getSmplIndex" is a type-bound procedure on a data object of type(DdsObject). This procedure returns the index to the first sample in this data object's file. The index is computed in terms of the host-dependent sample type associated with this data object.

EXAMPLE

	use DdsIO_mod, only : DdsObject
	use Precision_mod, only : DPI
	
	type(DdsObject)  :: inData
	integer(kind=DPI) :: nhdrs

	call inData%setAttr('passhdrs',.true.)
	call inData%open('r', 'in')
	nhdrs = inData%getSmplIndex() - 1

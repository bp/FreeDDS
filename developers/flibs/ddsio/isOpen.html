isOpen - Tests whether the data object has been opened

SYNOPSIS

use DdsIO_mod

logical function isOpen()

DESCRIPTION

"isOpen" is a type-bound procedure on a data object of type(DdsObject). This procedure returns whether or not this data object has both been opened with data%open and not been closed with data%close.

EXAMPLE

	use DdsIO_mod, only : DdsObject
	use dds, only : fdds_prterr
	
	type(DdsObject) :: inData
	integer         :: ier
	
	! Test whether input data is opened in read mode
	call inData%open('r','in')
	if (.not. inData%isOpen() ) then
	    ier = fdds_prterr('Opening in!\n\0')
	    stop
	endif
	
	! close data
	call inData%close()

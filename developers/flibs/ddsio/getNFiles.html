getNFiles - Returns the number of files associated with this data object

SYNOPSIS

use DdsIO_mod

integer function getNFiles()

DESCRIPTION

"getNFiles" is a type-bound procedure on a data object of type(DdsObject). This procedure returns the number of files associated with this data object.

EXAMPLE

	use DdsIO_mod, only : DdsObject
	
	type(DdsObject) :: inData
	integer         :: inFiles
	
	call inData%open('r', 'in')
	inFiles = inData%getNFiles()

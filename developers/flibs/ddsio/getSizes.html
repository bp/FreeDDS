getSizes - Returns this data object's sizes

SYNOPSIS

use DdsIO_mod

integer(kind=DPI), dimension(:) function getSizes(ifile)

DESCRIPTION

"getSizes" is a type-bound procedure on a data object of type(DdsObject). This procedure returns the sizes of the dimensions of this data object. If multiple files are merged along a specified axis, then the merged axis size is properly computed.

If ifile is specified, then this procedure returns the sizes of the given file of this data object.

EXAMPLE

	use DdsIO_mod, only : DdsObject
	use Precision_mod, only : DPI
	
	type(DdsObject)   :: inData
	integer(kind=DPI) :: inSizes(3)
	
	call inData%open('r', 'in')
	inSizes = inData%getSizes()

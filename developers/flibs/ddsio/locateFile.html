locateFile - Returns the file number corresponding to a position in the merged axis

SYNOPSIS

use DdsIO_mod

integer function locateFile(k)

integer, intent(in) :: k

DESCRIPTION

"locateFile" is a type-bound procedure on a data object of type(DdsObject). This procedure, using the current index set of this data object, returns the file number of the file that contains index k in the merged axis.

EXAMPLE

	use DdsIO_mod, only : DdsObject
	use dds, only : fdds_prtcon
	
	type(DdsObject)   :: inData
	integer           :: indexOfInterest, iFile, ier
	
	indexOfInterest = 100
	
	call inData%open('r', 'in')
	iFile = inData%locateFile(indexOfInterest)
	ier = fdds_prtcon('merged axis index %d is located in file %d\n\0', indexOfInterest, iFile)

getDeltas - Returns this data object's deltas

SYNOPSIS

use DdsIO_mod

real(kind=DPR), dimension(:) function getDeltas()

DESCRIPTION

"getDeltas" is a type-bound procedure on a data object of type(DdsObject). This procedure returns the deltas for each dimension of this data object.

EXAMPLE

	use DdsIO_mod, only : DdsObject
	use Precision_mod, only : DPR
	
	type(DdsObject) :: inData
	real(kind=DPR)  :: inDeltas(3)
	
	call inData%open('r', 'in')
	inDeltas = inData%getDeltas()

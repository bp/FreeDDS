getAxes - Returns this data object's axes names

SYNOPSIS

use DdsIO_mod

character(len=*), dimension(:) function getAxes()

DESCRIPTION

"getAxes" is a type-bound procedure on a data object of type(DdsObject). This procedure returns the axes names associated with this data object.

EXAMPLE

	use DdsIO_mod, only : DdsObject
	
	type(DdsObject) :: inData
	character(len=AXISNAME_MAX) :: axes(3)
	
	call inData%open('r', 'in')
	axes = inData%getAxes()

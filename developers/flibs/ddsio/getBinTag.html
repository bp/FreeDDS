getBinTag - Returns this data object's binary tag

SYNOPSIS

use DdsIO_mod

integer function getBinTag(ifile)

integer, optional, intent(in) :: ifile

DESCRIPTION

"getBinTag" is a type-bound procedure on a data object of type(DdsObject). This procedure returns the data object's associated binary tag. If ifile is specified, then this procedure returns the binary tag of file ifile.

EXAMPLE

	use DdsIO_mod, only : DdsObject
	
	type(DdsObject) :: inData
	integer         :: binTag
	
	call inData%open('r', 'in')
	binTag = inData%getBinTag()

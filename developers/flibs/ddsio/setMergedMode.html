setMergedMode - Specifies how multiple files are merged

SYNOPSIS

use DdsIO_mod

procedure setMergedMode(mode)

integer, intent(in) :: mode

DESCRIPTION

"setMergedMode" is a type-bound procedure on a data object of type(DdsObject). For a data object associated with multiple files in read mode, this procedure specifies how the axes should be merged across files.

If mode is 0, then the files are merged across the specified axis. All axes except the merged axis must be conformant. If mode is 1, then all axes including the merged axis must be conformant so that subblocks subblocks can be read across multiple files.

EXAMPLE

	use DdsIO_mod, only : DdsObject
	
	type(DdsObject) :: inData
	
	! merge files across 2nd axis
	call inData%setMergedMode(0)
	call inData%setMergedAxis(2)
	
	call inData%open('r', 'in')

setAttr - Sets an attribute for this data object

SYNOPSIS

use DdsIO_mod

procedure setAttr(key, val)

character(len=*), intent(in) :: key
intent(in) :: val

DESCRIPTION

"setAttr" is a type-bound procedure on a data object of type(DdsObject). This procedure sets the attribute labeled by key as val for this data object. Note that the type of val depends on which key is specified. data%setAttr should be called before data%open.

'seekopen' logical specifying that the file in write mode should be forced open with a seek when data%open is called.
'passhdrs' logical specifying that trace headers should also be read from file.
'zip' logical specifying that file in write mode should have format asp zip.
'title' character(len=*) specifying the title used to open file through fddx_out or fddx_in.
'axes' character(len=*) specifying the space-delimited axes of the file in write mode.
'smpltype' character(len=*) specifying the host-dependent sample type of the input/output file: 'short', 'int', 'float', or 'complex'. Can be set to 'FROM_FILE' for read mode.
'datapath' character(len=*) specifying the data path of the file in write mode.
'axesnames' character(len=*), dimension(:) specifying the axes names of the file(s) in read mode.
'units' character(len=*), dimension(:) specifying the units of the file in write mode.
'rank' integer specifying the dimensions of the file(s).
'size' integer, dimension(:) specifying the size of each axis of the file(s).
'delta' real, dimension(:) specifying the delta of each axis of the file(s).
'origin' real, dimension(:) specifying the origin of each axis of the file(s).

EXAMPLE

	use DdsIO_mod, only : DdsObject
	
	type(DdsObject) :: outData
	
	! set attributes for output data
	call outData%setAttr('rank', 4)
	call outData%setAttr('axes', 'z x y t')
	call outData%setAttr('size', [100, 100, 100, 25])
	call outData%setAttr('delta', [25.0, 25.0, 25.0, .001])
	call outData%setAttr('origin', [0, 0, 0, 0])
	call outData%setAttr('units', ['m', 'm', 'm', 'sec'])
	call outData%setAttr('seekopen', .true.)
	
	! open output data in write mode
	call outData%open('w', 'out')
	
	! close data
	call outData%close()

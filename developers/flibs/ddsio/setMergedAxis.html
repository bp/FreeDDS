setMergedAxis - Specifies which axis should be merged across multiple files

SYNOPSIS

use DdsIO_mod

procedure setMergedAxis(k)

integer, intent(in) :: k

DESCRIPTION

"setMergedAxis" is a type-bound procedure on a data object of type(DdsObject). For a data object associated with multiple files in read mode, this procedure specifies which axis should be merged across multiple files.

EXAMPLE

	use DdsIO_mod, only : DdsObject
	
	type(DdsObject) :: inData
	
	! merge files across 2nd axis
	call inData%setMergedMode(0)
	call inData%setMergedAxis(2)
	
	call inData%open('r', 'in')

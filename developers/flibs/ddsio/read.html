read - Reads in data from the file(s) into a given buffer

SYNOPSIS

use DdsIO_mod

procedure read(buffer)

dimension(*), intent(out) :: buffer

DESCRIPTION

"read" is a type-bound procedure on a data object of type(DdsObject). This procedure reads in data from the file so as to fill the buffer, starting from the trace offset specified by the seekTo procedure. This procedure can only be called on a data object that has been opened in mode 'r'.

"read" is a generic function written for a buffer of dimension: 1-7, and type: 1-byte integer, 2-byte integer, 4-byte integer, 8-byte integer, 4-byte real, 8-byte real, 8-byte complex, or 16-byte complex.

EXAMPLE

	use DdsIO_mod, only : DdsObject
	use Precision_mod, only : DPI
	
	type(DdsObject)   :: inData, in2Data
	integer(kind=DPI), allocatable :: inSizes(:)
	real, allocatable :: buffer(:,:,:), buffer2(:,:,:)
	
	allocate(inSizes(3))
	
	! open data in read mode without headers
	call inData%open('r','in')
	inSizes = inData%getSizes()
	allocate(buffer(inSizes(1), inSizes(2), inSizes(3)))
	
	! open data2 in read mode with headers
	call in2Data%setAttr('passhdrs',.true.)
	call in2Data%open('r','in2')
	inSizes = in2Data%getSizes()
	traceSize = in2Data%getTraceSize()
	allocate(buffer2(traceSize, inSizes(2), inSizes(3))
	
	! read entire 3D volumes
	call inData%read(buffer)
	call in2Data%read(buffer2)
	
	! close data
	call inData%close()
	call in2Data%close()
	deallocate(inSizes)

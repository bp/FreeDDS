getOrigins - Returns this data object's origins

SYNOPSIS

use DdsIO_mod

real(kind=DPR), dimension(:) function getOrigins()

DESCRIPTION

"getOrigins" is a type-bound procedure on a data object of type(DdsObject). This procedure returns the origins for each dimension of this data object.

EXAMPLE

	use DdsIO_mod, only : DdsObject
	use Precision_mod, only : DPR
	
	type(DdsObject) :: inData
	real(kind=DPR)  :: inOrigins(3)
	
	call inData%open('r', 'in')
	inOrigins = inData%getOrigins()

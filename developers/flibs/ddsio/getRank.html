getRank - Returns this data object's rank

SYNOPSIS

use DdsIO_mod

integer function getRank()

DESCRIPTION

"getRank" is a type-bound procedure on a data object of type(DdsObject). This procedure returns the rank of this data object.

EXAMPLE

	use DdsIO_mod, only : DdsObject
	
	type(DdsObject) :: inData
	integer         :: inRank
	
	call inData%open('r', 'in')
	inRank = inData%getRank()

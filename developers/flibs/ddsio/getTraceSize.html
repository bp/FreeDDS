getTraceSize - Returns the size of this data object's trace including headers

SYNOPSIS

use DdsIO_mod

integer(kind=DPI) function getTraceSize()

DESCRIPTION

"getTraceSize" is a type-bound procedure on a data object of type(DdsObject). This procedure returns the size of this data object's trace (including trace headers). The size is computed in terms of the host-dependent sample type associated with this data object.

EXAMPLE

	use DdsIO_mod, only : DdsObject
	use Precision_mod, only : DPI
	
	type(DdsObject)   :: inData
	integer(kind=DPI) :: traceSize
	real, allocatable :: trace(:)

	call inData%setAttr('passhdrs',.true.)
	call inData%open('r', 'in')
	traceSize = inData%getTraceSize()
	allocate(trace(traceSize))

setSizes - Sets this data object's sizes for multiple files

SYNOPSIS

use DdsIO_mod

procedure setSizes(nsize, irank, ifile)

integer(kind=DPI), intent(in) :: nsize(:)
integer, optional, intent(in) :: irank
integer, optional, intent(in) :: ifile

DESCRIPTION

"setSizes" is a type-bound procedure on a data object of type(DdsObject). This procedure sets the size of each rank of each file associated with this data object using nsize(:).

If irank is specified, then only the first element of nsize is used to set the size of rank irank.

If ifile is not specified, this procedure sets the sizes for all files with a merged axis. If ifile is specified and within the range of files associated with this data object, this procedure only sets the size of the single file indexed by ifile.

NOTE: "setSizes" only effects files in read mode. To set the size of a file in write mode, use data%setAttr('size', sizes).

EXAMPLE

	use DdsIO_mod, only : DdsObject
	
	type(DdsObject) :: inData
	
	call inData%setSizes([100, 100, 100])
	
	call inData%open('r', 'in')

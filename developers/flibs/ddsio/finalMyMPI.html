finalMyMPI - Finalizes MPI for a program

SYNOPSIS

use MyMPI_mod

subroutine finalMyMPI()

DESCRIPTION

"finalMyMPI" finalizes MPI for both the MyMPI_mod module and the program itself. Internally, this subroutine calls mpi_finalize.

EXAMPLE

	program test
	
	    use MyMPI_mod
	    use dds, only : fdds_prtcon
	
	    integer :: ier
	
	    call initMyMPI()
	
	    ier = fdds_prtcon('rank= %d, np= %d, nproc= %d\n\0', myid, nodes, nproc)
	
	    call finalMyMPI()
	
	end program test

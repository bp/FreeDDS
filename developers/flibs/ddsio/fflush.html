fflush - Flushes all or a selected binary stream

SYNOPSIS

use DdsIO_mod

procedure fflush()

DESCRIPTION

"fflush" is a type-bound procedure on a data object of type(DdsObject). This procedure internally uses fdds_flush to flush the binary tag associated with this data object. If the binary tag has not been set (or is set as -1), then all binary streams are flushed.

EXAMPLE

	use DdsIO_mod, only : DdsObject
	
	type(DdsObject) :: inData
	
	call inData%open('r', 'in')
	call inData%fflush()

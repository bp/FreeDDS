getType - Returns this data object's host-dependent sample type

SYNOPSIS

use DdsIO_mod

character(len=*) function getType()

DESCRIPTION

"getType" is a type-bound procedure on a data object of type(DdsObject). This procedure returns the host-dependent sample type associated with this data object. Possible return values are 'short', 'int', 'float', or 'complex'.

EXAMPLE

	use DdsIO_mod, only : DdsObject
	use DDS, only : DEFNNAME_MAX
	
	type(DdsObject)   :: inData
	character(len=DEFNNAME_MAX) :: sampleType
	
	! Allows input file to determine sample type
	call inData%setAttr('smpltype','FROM_FILE')
	call inData%open('r', 'in')
	
	! Accesses the sample type of the input file
	sampleType = inData%getType()

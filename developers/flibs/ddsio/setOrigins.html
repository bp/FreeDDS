setOrigins - Sets this data object's origins

SYNOPSIS

use DdsIO_mod

procedure setOrigins(origin, irank)

real(kind=DPR), intent(in) :: origin(:)
integer, optional, intent(in) :: irank

DESCRIPTION

"setOrigins" is a type-bound procedure on a data object of type(DdsObject). This procedure sets the origins of each rank of each file associated with this data object using origin(:).

If irank is specified, then only the first element of origin is used to set the origin of rank irank.

NOTE: "setOrigins" only effects files in read mode. To set the size of a file in write mode, use data%setAttr('origin', origins).

EXAMPLE

	use DdsIO_mod, only : DdsObject
	
	type(DdsObject) :: inData
	
	call inData%setOrigins([0.0, 0.0, 0.0])
	
	call inData%open('r', 'in')

setDeltas - Sets this data object's deltas

SYNOPSIS

use DdsIO_mod

procedure setDeltas(delta, irank)

real(kind=DPR), intent(in) :: delta(:)
integer, optional, intent(in) :: irank

DESCRIPTION

"setDeltas" is a type-bound procedure on a data object of type(DdsObject). This procedure sets the deltas of each rank of each file associated with this data object using delta(:).

If irank is specified, then only the first element of delta is used to set the delta of rank irank.

NOTE: "setDeltas" only effects files in read mode. To set the size of a file in write mode, use data%setAttr('delta', deltas).

EXAMPLE

	use DdsIO_mod, only : DdsObject
	
	type(DdsObject) :: inData
	
	call inData%setDeltas([25.0, 25.0, 25.0])
	
	call inData%open('r', 'in')

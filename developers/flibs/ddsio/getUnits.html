getUnits - Returns this data object's unit names

SYNOPSIS

use DdsIO_mod

character(len=*), dimension(:) function getUnits()

DESCRIPTION

"getUnits" is a type-bound procedure on a data object of type(DdsObject). This procedure returns the unit names for each dimension of this data object.

EXAMPLE

	use DdsIO_mod, only : DdsObject
	use DDS, only : UNITNAME_MAX
	
	type(DdsObject)             :: inData
	character(len=UNITNAME_MAX) :: inUnits(3)
	
	call inData%open('r', 'in')
	inUnits = inData%getUnits()

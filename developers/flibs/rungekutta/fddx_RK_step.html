fddx_RK_step - take a step using a Runge-Kutta extrapolator.

SYNOPSIS

int fddx_RK_step(rk_tag, dt, yin, dydt, yout, dyout)

integer rk_tag
real dt
real yin(*)
real dydt(*)
real yout(*)
real dyout(*)

DESCRIPTION

fddx_RK_step takes a step using Runge-Kutta extrapolator.
NOTE that this function is not re-entrant, which means that you cannot use it inside an OpenMP parallelized loop.
Each thread must use a separate extrapolator, created using fddx_RK_new.

INPUT

The parameter, "rk_tag", specifies which Runge-Kutta extrapolator to show. Use function fddx_RK_new to create the extrapolator.

The parameter, "dt", specifies the step length.

The parameter, "yin", specifies the input position vector.

The parameter, "dydt", specifies the input direction vector.

The parameter, "yout", specifies the input position vector.

The parameter, "dyout", specifies the input direction vector.

RETURN VALUE

fddx_RK_step returns 0 for normal termination, or -1 if there was an error.

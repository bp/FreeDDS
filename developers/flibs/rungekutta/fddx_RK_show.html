fddx_RK_show - write summary details of a Runge-Kutta extrapolator to the print file.

SYNOPSIS

integer fddx_RK_show(rk_tag)

integer rk_tag

DESCRIPTION

fddx_RK_show writes summary details of a Runge-Kutta extrapolator to the print file.

INPUT

The parameter, "rk_tag", specifies which Runge-Kutta extrapolator to show. Use function fddx_RK_new to create the extrapolator.

RETURN VALUE

fddx_RK_show always returns 0.

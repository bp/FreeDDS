fddx_RK_new - create a new Runge-Kutta extrapolator.

SYNOPSIS

integer fddx_RK_new(alias, nn, RHS)

character*(*)alias
integer nn
extern integer (*RHS)(integer nn, real vec_in(nn), real vec_out(nn) )

DESCRIPTION

fddx_RK_new creates a new Runge-Kutta extrapolator.
It scans for user input to decide which type of Runge-Kutta method to use.
Currently implemented methods are 2nd (rk2) and 4th order (rk4).
The user can specify which method to use by specifying rk=rk2 or rk=rk4.

INPUT

The parameter, "alias", specifies a prefix to add to the rk= keyword to allow the use of multiple different Runge-Kutta methods with the same application.

The parameter, "nn", specifies the maximum size of the vector to propagate.

The parameter, "RHS", specifies a function to compute the Right Hand Size of the equations.

RETURN VALUE

fddx_RK_new returns a tag to the newly created Runge-Kutta Extrapolator.
In the event of an error, it will return -1.

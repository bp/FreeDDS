fddz_scann - Get the name of a definition found by fdds_scan*()

SYNOPSIS

#include "fdds.h"
#include "fddz.h"

integer function fddz_scann(alias, defnname)

character alias*(*)
character defnname*(*)

DESCRIPTION

Function "fddz_scann" finds the "name" of the definition that is found by any of the fdds_scan*() functions. This is used to give applications a way around the "single alias name" restriction of convenience routines fddx_in*, fddx_mod*, or fddx_out*.

OUTPUT

When a definition is found, defnname is set to the "name" part of the definition and the length of the the name is returned.

When no definition is found, or if no dictionaries are in scan mode, defnname is set to NULL and EOF (-1) is returned.

EXAMPLE

integer   ier, fd_in
character defnname*(DEFNNAME_MAX), Title*(*)
 ...
c
c Use convenience routines to open input with more than one alias
c
ier = fddz_scann('Vp Vz vel',defnname)
fd_in = fddx_in (defnname,'stdin:',Title)

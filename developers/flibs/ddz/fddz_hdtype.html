fddz_hdtype - Detect the host-dependent type of a field

SYNOPSIS

#include "fdds.h"
#include "fddz.h"

integer function fddz_hdtype(host_type, bin_tag, field_tag)

character host_type*(*)
integer bin_tag
integer field_tag

DESCRIPTION

Function "fddz_hdtype" returns the host-dependent type for a field. Host-dependent type is uniquely determined by the combination of "genus" and "precision". The table below shows the association between genus, precision, and host-dependent type. 
                           bytes of
genus     host_type        precision   FORTRAN type       note

integer   signed char          1       N/A                 #
integer   short                2       integer*2           +
integer   int                  4       integer
integer   long long            8       integer*8           +
 
float     float                4       real
float     double               8       double precision
float     long double         16       real*16             +

complex   complex              8       complex
complex   complex double      16       double complex      +
complex   complex long        32       N/A                 #

+ not part of F77 specification - often available anyway
# not part of F77 specification - no implementations known to author

INPUT

bin_tag is a binary tag, previously obtained from cdds_open. It specifies a particular binary stream. field_tag is the tag of the field for which the host-dependent type is desired. It can be found with "fdds_member".

OUTPUT

A value of -1 is returned on any error.

EXAMPLE

integer bin, tag, ier
character RecNum_type*(*), Sample_type*(*)
 ...
c host_type of RecNum field
tag = fdds_member(bin, 0, "RecNum")
ier = fddz_hdtype(RecNum_type, bin, tag)
 ...
c host_type of Samples field
tag = cdds_member(bin, 0, "Samples")
ier = fddz_hdtype(Sample_type, bin, tag)

fddz_title - Print the standard DDS title block to a dictionary in print mode

SYNOPSIS

#include "fdds.h"
#include "fddz.h"

integer function fddz_title(Title)

character Title*(*)

DESCRIPTION

Function "fddz_title" gives FORTRAN programs the ability to print the standard DDS title block to dictionaries that are opened without any history information. An example of such a title is shown below. 
DDS_SENTRY= 231 2011-07-15 09:29:24 -0500 (Fri, 15 Jul 2011)
********************************************
******************* TITLE ******************
********************************************
cmd_title=  "editd: general seismic editting program"
cmd_name= /apps/global/dist/Linux/2.6/mbs/prod/x86_64/editd
cmd_rev= 7632
cmd_ddsrev= 231
cmd_user= zjwe06
cmd_date= Fri Sep 14 07:17:14 2012
cmd_host= hpcp8004
cmd_cwd= /bphpc6/vol0/zjwe06/test_data
cmd_pid= 8724

EXAMPLE

integer fd_tmp,ier
character Title*(*),tmp_dict*(*)
 ...
ier = fdds_out(tmp_dict,'tmp',' ',' ')
ier = fddz_title(Title)
 ...
fd_tmp=fdds_open(tmp_dict,' ',' ','w+')

fddz_md5sum - Get the md5sum of a buffer

SYNOPSIS

#include "fdds.h"
#include "fddz.h"

integer function fddz_md5sum(checksum, buffer, byte_count)

character checksum*(*)
real buffer(*)
integer*8 byte_count

DESCRIPTION

Function "fddz_md5sum" gives programs access to GNU core utils function md5_buffer. This is the message-digest function used by utility md5sum(1). Access to the MD5 algorithm is useful when trying to locate system bugs that are causing headaches.

INPUT

buffer is the memory block to digest. byte_count tells "fddz_md5sum" the number of bytes to process.

OUTPUT

The result is returned in variable checksum. Variable checksum should be declaired with a length of at least 33 to hold the entire result. Shorter character variables will work, but resolution will be lost.

On error, -1 is returned.

SEE ALSO

   md5sum(1) compute and check MD5 message digest

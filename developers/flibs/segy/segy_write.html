segy_write - write a segy trace

SYNOPSIS

subroutine segy_write(fd, buf, nsmp)

integer fd
real(*) buf
integer nsmp

DESCRIPTION

The routine, "segy_write", writes a trace to the segy file identified by the file descriptor, fd, returned from "segy_out". The array, buf must contain the segy trace header in the first 240 bytes followed by nsmp number of data samples. The return value will be "1" if successful; otherwise "0" is returned.

segy_read - read a segy trace

SYNOPSIS

integer segy_read(fd, buf, nsmp)

integer fd
real(*) buf
integer nsmp

DESCRIPTION

The routine, "segy_read", reads a trace from the segy f file as identified by the file descriptor, fd, into the array, buf (which will contain 240 bytes of trace header followed by nsmp data samples. The return value will be "0" if an error is encountered and "1" if successfully read.

segy_in - open a segy file for reading

SYNOPSIS

integer segy_in(filename, buf, nsmp, ntrc)

character*(*) filename
real(900) buf
integer nsmp
integer ntrc

DESCRIPTION

The routine, "segy_in", opens a segy file named, filename, for reading. The segy line header is returned into the array, buf. The number of samples/trace and the number of traces/record are returned in nsmp and ntrc. The return value is the file descriptor which must be used in the "segy_read" routine.

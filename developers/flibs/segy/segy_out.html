segy_out - open a segy file for writing

SYNOPSIS

integer segy_out(filename, buf, nsmp, ntrc)

character*(*) filename
real(900) buf
integer nsmp
integer ntrc

DESCRIPTION

The routine, "segy_out", creates an output segy file with the name, filename. The buf must contain the segy line header, nsmp must have the number of samples in each trace, and ntrc must be the number of traces/record. The returned value is the file descriptor which can be used in the segy_write routine.

segy_init - initialize the segy I/O interface datatype

SYNOPSIS

integer segy_init(n)

integer n

DESCRIPTION

The routine, "segy_init", sets the segy Samples data type. If this is not called, the default will be ibm floating point (1). The return value is always "1".

  • 1: ibm floating point
  • 2: ieee floating point (big endian)
  • 3: ieee floating point (little endian)

hsrti2 - Sort integer array

SYNOPSIS

subroutine hsrti2(n, x, y)

integer n
integer(*) x
integer(*) y

DESCRIPTION

The routine, "hsrti2", sorts the integer array, x, of length n into ascending numerical order using the Heapsort algorithm, while making the corresponding rearrangement of the integer array y. This is identical to the routine, "isort2".

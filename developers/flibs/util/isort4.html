isort4 - Sort integer array

SYNOPSIS

subroutine isort4(n, x, y, z, t)

integer n
integer(*) x
integer(*) y
integer(*) z
integer(*) t

DESCRIPTION

The routine, "isort4", sorts the integer array, x, of length n into ascending numerical order using the Heapsort algorithm, while making the corresponding rearrangement of the integer array y, z and t.

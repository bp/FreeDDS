hsort4 - Sort real array

SYNOPSIS

subroutine hsort4n, x, y, z, t)

integer n
real(*) x
real(*) y
real(*) z
real(*) t

DESCRIPTION

The routine, "hsort4", sorts the real array, x, of length n into ascending numerical order using the Heapsort algorithm, while making the corresponding rearrangement of the real arrays y, z and t.

fddx_RT_ttime - calculate the traveltime along a segment of a ray.

SYNOPSIS

#include <fddx_raytracer.h>

real fddx_RT_ttime(rt_tag, ds0, dt, point0, gpvel0, point1, gpvel1)

integer rt_tag
real ds0
real dt
real point0(3)
real gpvel0(3)
real point1(3)
real gpvel1(3)

DESCRIPTION

fddx_RT_ttime calculates the traveltime along a segment of a ray from point0 to point1.

INPUT

The parameter, "rt_tag", specifies which raytracer to use.
Use function fddx_RT_new to create the raytracer.

The parameter, "ds0", specifies the ray step length.

The parameter, "dt", specifies the traveltime from point0 to point1 using the smooth velocity.

The parameter, "point0", specifies the start point of the segment.

The parameter, "gpvel0", specifies the group velocity at start point of the segment.

The parameter, "point1", specifies the end point of the segment.

The parameter, "gpvel1", specifies the group velocity at end point of the segment.

RETURN VALUE

fddx_RT_ttime returns the traveltime in the (non-smooth) velocity for the ray to propagate from point0 to point1.

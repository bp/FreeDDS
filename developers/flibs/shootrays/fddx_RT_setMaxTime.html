fddx_RT_setMaxTime - set the maximum traveltime of ray before the raytracer stops.

SYNOPSIS

#include <fddx_raytracer.h>

real fddx_RT_setMaxTime(rt_tag, maxtime)

integer rt_tag
real maxtime

DESCRIPTION

fddx_RT_setMaxTime sets the maximum traveltime of ray before the raytracer stops.

INPUT

The parameter, "rt_tag", specifies for which raytracer to return the maximum traveltime.

The parameter, "maxtime", specifies the maximum traveltime.

RETURN VALUE

fddx_RT_setMaxTime returns the maximum traveltime of ray before the raytracer stops.

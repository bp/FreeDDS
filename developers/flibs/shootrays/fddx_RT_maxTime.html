fddx_RT_maxTime - returns the maximum traveltime of ray before the raytracer stops.

SYNOPSIS

#include <fddx_raytracer.h>

real fddx_RT_maxTime(rt_tag)

integer rt_tag

DESCRIPTION

fddx_RT_maxTime returns the maximum traveltime of ray before the raytracer stops.

INPUT

The parameter, "rt_tag", specifies for which raytracer to return the maximum traveltime.

RETURN VALUE

fddx_RT_maxTime returns the maximum traveltime of ray before the raytracer stops.

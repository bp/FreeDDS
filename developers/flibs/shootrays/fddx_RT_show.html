fddx_RT_show - write summary details of a raytracer to the print file.

SYNOPSIS

#include <fddx_raytracer.h>

integer fddx_RT_show(rt_tag)

integer rt_tag

DESCRIPTION

fddx_RT_show writes summary details of a raytracer to the print file.

INPUT

The parameter, "rt_tag", specifies which raytracer to show. Use function fddx_RT_new to create the raytracer.

RETURN VALUE

fddx_RT_show always returns 0.

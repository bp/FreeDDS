fddx_RT_stepLength - returns the step length that the raytracer should take.

SYNOPSIS

#include <fddx_raytracer.h>

real fddx_RT_stepLength(rt_tag)

integer rt_tag

DESCRIPTION

fddx_RT_stepLength returns step length that the raytracer should take.

INPUT

The parameter, "rt_tag", specifies for which raytracer to return the step length.

RETURN VALUE

fddx_RT_stepLength returns the step length that the raytracer should take.

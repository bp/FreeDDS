fddx_RT_RHS - calculate the Right Hand Side (RHS) of the raytracing equations.

SYNOPSIS

#include <fddx_raytracer.h>

integer fddx_RT_RHS(rt_tag, nn, vec_in, vec_out)

integer rt_tag
integer nn
real vec_in(nn)
real

DESCRIPTION

fddx_RT_RHS calculates the right hand side of the raytracing equations.

INPUT

The parameter, "rt_tag", specifies for which raytracer to renormalize the point.

The parameter, "nn", specifies the lengths of the input and output vectors.

The parameter, "vec_in", specifies the input vector.

The parameter, "vec_out", specifies the output vector.

RETURN VALUE

fddx_RT_RHS returns 0 unless there is an error, in which case it returns -1.

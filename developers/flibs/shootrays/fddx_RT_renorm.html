fddx_RT_renorm - renormalize the ray vector to satisfy U.p=1.

SYNOPSIS

#include <fddx_raytracer.h>

integer fddx_RT_renorm(rt_tag, vec_in, vec_out)

integer rt_tag
real vec_in
real vec_out

DESCRIPTION

fddx_RT_renorm normalizes the ray vector to satisfy U.p=1,
where U is the group velocity vector and p is the slowness vector.

INPUT

The parameter, "rt_tag", specifies for which raytracer to renormalize the point.

The parameter, "vec_in", specifies the input vector.

The parameter, "vec_out", specifies the output vector.

RETURN VALUE

fddx_RT_renorm returns 0 unless there is an error, in which case it returns -1.

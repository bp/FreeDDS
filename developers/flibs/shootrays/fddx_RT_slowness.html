fddx_RT_slowness - compute the slowness vector.

SYNOPSIS

#include <fddx_raytracer.h>

integer fddx_RT_slowness(rt_tag, vec_in, vec_out)

integer rt_tag
real vec_in(6)
real vec_out(3)

DESCRIPTION

fddx_RT_slowness calculates the slowness vector.

INPUT

The parameter, "rt_tag", specifies which raytracer to show. Use function fddx_RT_new to create the raytracer.

The parameter, "vec_in", specifies the input vector of location and direction.

The parameter, "vec_out", specifies the output slowness vector.

RETURN VALUE

fddx_RT_slowness always returns 0, unless there is an error, in which case it returns -1.

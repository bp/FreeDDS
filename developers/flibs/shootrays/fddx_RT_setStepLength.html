fddx_RT_setStepLength - set the step length that the raytracer should take.

SYNOPSIS

#include <fddx_raytracer.h>

real fddx_RT_setStepLength(rt_tag, length)

integer rt_tag
real length

DESCRIPTION

fddx_RT_setStepLength sets the step length that the raytracer should take.

INPUT

The parameter, "rt_tag", specifies for which raytracer to return the step length.

The parameter, "length", specifies the step length.

RETURN VALUE

fddx_RT_setStepLength returns the step length that the raytracer should take.

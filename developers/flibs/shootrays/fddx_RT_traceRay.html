fddx_RT_traceRay - trace a ray.

SYNOPSIS

#include <fddx_raytracer.h>

integer fddx_RT_traceRay(rt_tag, xs, ys, zs, decl, azm, output)

integer rt_tag
real xs, ys, zs
real decl
real azm
real output(*)

DESCRIPTION

fddx_RT_traceRay traces a ray.

INPUT

The parameter, "rt_tag", specifies which raytracer to show.
Use function fddx_RT_new to create the raytracer.

The parameter, "xs, ys, zs", specifies the source location.

The parameter, "decl", specifies the ray declination.

The parameter, "azm", specifies the ray azimuth.

OUTPUT

The parameter, "output", specifies the array for the output parameters.
Note that output should be of sufficient size.
See the function fddx_requestOut for defining which parameters to output.

RETURN VALUE

fddx_RT_traceRay always returns 0, unless there is an error, in which case it returns -1.

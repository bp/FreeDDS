fddx_RT_setNumSteps - set the number steps of the ray.

SYNOPSIS

#include <fddx_raytracer.h>

integer fddx_RT_setNumSteps(rt_tag, numsteps)

integer rt_tag
integer numsteps

DESCRIPTION

fddx_RT_setNumSteps sets the number of steps of the ray.

INPUT

The parameter, "rt_tag", specifies for which raytracer to return the number of steps.

The parameter, "numsteps", specifies the number of steps.

RETURN VALUE

fddx_RT_setNumSteps returns the number of steps of the ray.

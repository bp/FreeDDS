fddx_RT_numSteps - returns the number steps of the ray.

SYNOPSIS

#include <fddx_raytracer.h>

integer fddx_RT_numSteps(rt_tag)

integer rt_tag

DESCRIPTION

fddx_RT_numSteps returns the number of steps of the ray.

INPUT

The parameter, "rt_tag", specifies for which raytracer to return the number of steps.

RETURN VALUE

fddx_RT_numSteps returns the number of steps of the ray.

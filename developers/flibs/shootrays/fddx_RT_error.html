fddx_RT_error - returns the raytracer error status.

SYNOPSIS

#include <fddx_raytracer.h>

integer fddx_RT_error(rt_tag)

integer rt_tag

DESCRIPTION

fddx_RT_error returns the raytracer error status.

INPUT

The parameter, "rt_tag", specifies for which raytracer to return the error stauts.

RETURN VALUE

fddx_RT_error always returns 0.

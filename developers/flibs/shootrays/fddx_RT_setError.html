fddx_RT_setError - set the raytracer error status.

SYNOPSIS

#include <fddx_raytracer.h>

integer fddx_RT_setError(rt_tag, error)

integer rt_tag
integer error

DESCRIPTION

fddx_RT_setError sets the raytracer error status.

INPUT

The parameter, "rt_tag", specifies for which raytracer to set the status.

The parameter, "error", specifies the new error status value.

RETURN VALUE

fddx_RT_setError always returns the error status, error.

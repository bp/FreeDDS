pfddx_userrange - scan the parameter file and command line for editd style dimension parameters.

SYNOPSIS

#include <pfddx.h>

integer fddx_userrange(node, rank, size, ss, ee, ii)

integer node
integer rank
integer size(RANK_MAX)
integer ss(RANK_MAX)
integer ee(RANK_MAX)
integer ii(RANK_MAX)

DESCRIPTION

fddx_userrange scans the parameter file and command line for editd style dimension parameters: 2s= 2e= 2i=, etc.
MPI node zero will also write a summary to the printout file.

INPUT

The parameter, "node", specifies the MPI node number.

The parameter, "rank", specifies the rank (number of dimensions) of the file.

The parameter, "size", specifies the size of each dimension.

OUTPUT

The parameter, "ss", specifies the user specified starting value to process for each axis.

The parameter, "ee", specifies the user specified end value to process for each axis.

The parameter, "ii", specifies the user specified increment to process for each axis.

RETURN VALUE

fddx_userrange always returns 0.

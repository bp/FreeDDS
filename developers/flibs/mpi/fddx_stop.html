fddx_stop - stop gracefully when using DDS and MPI

SYNOPSIS

#include <pfdds.h>

integer function fddx_stop(np, node)

integer np
integer node

DESCRIPTION

The function, "fddx_stop", sums all the errors reported by the different MPI threads through fdds_prterr. It then stops the program gracefully, stopping all MPI threads. Note that fddx_stop should be called by all MPI threads.

INPUT

The parameter, "np", specifies the number of MPI threads used by the program.

The parameter, "node", specifies the number of the current MPI thread.

RETURN VALUE

fddx_stop never returns.

fddx_initmpix - initialize MPI, and OpenMP, and start a print file or print on-line help.

SYNOPSIS

#include <pfdds.h>

integer function fddx_initmpix(nproc, np, node, prog, help)

integer nroc
integer np
integer node
character(*) prog
extern help

DESCRIPTION

The function, "fddx_initmpix", takes care of the generic initialisation needed by all DDS programs using MPI and/or OpenMP. It checks how many OpenMP and MPI threads were requested by the user. It checks if the user asked for the online help, and if so calls the function help and stops the program gracefully. Otherwise it opens the printout file.

INPUT

The parameter, "prog", specifies the program name. This information will be recorded in the printout file.

The parameter, "help", specifies the name of subroutine which writes the online help to the standard output.

OUTPUT

The parameter, "nproc", specifies the number of OpenMP threads used by the program.

The parameter, "np", specifies the number of MPI threads used by the program.

The parameter, "node", specifies the number of the current MPI thread.

RETURN VALUE

fddx_initmpix always returns 0, unless an error was detected, in which case the program aborts, and the function does not return.

fddx_writetrace - map headers and write traces.

SYNOPSIS

#include <pfddx.h>

integer fddx_writetrace(bin, buflen, nsamp, ntrace, buf, tr_tag, samples, numtags, tags, hvals)

integer bin
integer buflen
integer nsamp
integer ntrace
real buf(buflen,ntrace)
integer tr_tag
real samples(nsamp,ntrace)
integer numtags
integer tags(numtags)
float hvals(numtags,ntrace)

DESCRIPTION

fddx_writetrace inserts the samples and headers into the trace buffer and writes out the traces.

INPUT

The parameter, "bin", specifies the binary tag of the file.

The parameter, "buflen", specifies the length of the trace buffer.

The parameter, "nsamp", specifies the number of samples in the trace buffer.

The parameter, "ntrace", specifies the number of traces to write.

The parameter, "buf", specifies the trace buffer.

The parameter, "tr_tag", specifies the tag to the trace Samples.

The parameter, "samples", specifies the buffer for the samples.

The parameter, "numtags", specifies the number of header values to be inserted.

The parameter, "tags", specifies tags to the trace headers to be inserted.
The header values are only inserted if the tag values are >= 0.

The parameter, "hvals", specifies the buffer for the header values.

RETURN VALUE

fddx_writetrace returns the number traces that were successfully written to file.

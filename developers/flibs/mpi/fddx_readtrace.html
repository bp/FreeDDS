fddx_readtrace - read traces and map headers.

SYNOPSIS

#include <pfddx.h>

integer cddx_readtrace(bin, buflen, nsamp, ntrace, buf, tr_tag, samples, numtags, tags, hvals)

integer bin
integer buflen
integer nsamp
integer ntrace
real buf(buflen,ntrace)
integer tr_tag
real samples(nsamp,ntrace)
integer numtags
int tags(numtags)
realhvals(numtags,ntrace)

DESCRIPTION

fddx_readtrace reads ntrace traces from the input file.
Trace samples and specified headers are extraced.

INPUT

The parameter, "bin", specifies the binary tag of the file.

The parameter, "buflen", specifies the length of the trace buffer.

The parameter, "nsamp", specifies the number of samples in the trace buffer.

The parameter, "ntrace", specifies the number of traces to read.

The parameter, "tr_tag", specifies the tag to the trace Samples.

The parameter, "numtags", specifies the number of header values to be extracted.

The parameter, "tags", specifies tags to the trace headers to be extracted.
The header values are only extracted if the tag values are >= 0.

OUTPUT

The parameter, "buf", specifies the trace buffer.

The parameter, "samples", specifies the buffer for the samples.

The parameter, "hvals", specifies the buffer for the requested header values.

RETURN VALUE

fddx_readtrace returns the number traces that were successfully read from file.

fddx_checkforerrors - check for errors when running using DDS and MPI

SYNOPSIS

#include <fdds.h>

integer function fddx_checkforerrors(np, node)

integer np
integer node

DESCRIPTION

The function, "fddx_checkforerrors", sums all the errors reported by the different MPI threads through fdds_prterr. If there are errors on any of the threads then it aborts the program gracefully, stopping all threads.
Note that fddx_checkforerrors should be called by all MPI threads.

INPUT

The parameter, "np", specifies the number of MPI threads used by the program.

The parameter, "node", specifies the number of the current MPI thread.

OUTPUT

Always returns 0, unless an error was detected, in which case the program aborts, and the function does not return.

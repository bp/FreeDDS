fddx_V_SmoothVs - return a tag to a SmoothVs parameter.

SYNOPSIS

#include <fddx_vel.h>

integer fddx_V_SmoothVs(vel_tag)

integer vel_tag

DESCRIPTION

fddx_V_SmoothVs returns a tag to the SmoothVs parameter of a velocity.
If the velocity does not have SmoothVs defined, then fddx_V_SmoothVs returns -1.

INPUT

The parameter, "vel_tag", specifies a velocity for which to find the SmoothVs parameter.

RETURN VALUE

fddx_V_SmoothVs returns a tag to the SmoothVs parameter.
If SmoothVs is not defined, then it will return -1.

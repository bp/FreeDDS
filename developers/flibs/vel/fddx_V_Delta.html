fddx_V_Delta - return a tag to a Delta parameter.

SYNOPSIS

#include <fddx_vel.h>

integer fddx_V_Delta(vel_tag)

integer vel_tag

DESCRIPTION

fddx_V_Delta returns a tag to the Delta parameter of a velocity.
If the velocity does not have Delta defined, then fddx_V_Delta returns -1.

INPUT

The parameter, "vel_tag", specifies a velocity for which to find the Delta parameter.

RETURN VALUE

fddx_V_Delta returns a tag to the Delta parameter.
If Delta is not defined, then it will return -1.

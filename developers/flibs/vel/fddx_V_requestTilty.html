fddx_V_requestTilty - read a Tilty parameter.

SYNOPSIS

#include <fddx_vel.h>

integer fddx_V_requestTilty(vel_tag, critical)

integer vel_tag
integer critical

DESCRIPTION

fddx_V_requestTilty reads a Tilty parameter. This is is the tilt angle in the y-direction for TTI velocities.

INPUT

The parameter, "vel_tag", specifies a velocity with which to associate this Tilty parameter.
The keyword prefix for the users is "tilty". See also function fddx_V_newElement.

The parameter, "critical", specifies whether this parameter is critical.
If "critical" is non-zero and fddx_V_requestTilty is unable to find a suitable Tilty parameter, then a DDS error is printed.
NOTE that in the event of an error the program continues in order to allow for clean stopping of MPI codes.

RETURN VALUE

fddx_V_requestTilty returns a tag to the newly created Tilty parameter.
If it is unable to create Tilty, then it will return -1.

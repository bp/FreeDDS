fddx_V_requestVp2 - read a Vp squared parameter.

SYNOPSIS

#include <fddx_vel.h>

integer fddx_V_requestVp2(vel_tag, critical)

integer vel_tag
integer critical

DESCRIPTION

fddx_V_requestVp2 reads a Vp squared parameter.

INPUT

The parameter, "vel_tag", specifies a velocity with which to associate this Vp2 parameter.
The keyword prefix for the users is "vp2". See also function fddx_V_newElement.

The parameter, "critical", specifies whether this parameter is critical.
If "critical" is non-zero and fddx_V_requestVp2 is unable to find a suitable Vp2 parameter, then a DDS error is printed.
NOTE that in the event of an error the program continues in order to allow for clean stopping of MPI codes.

RETURN VALUE

fddx_V_requestVp2 returns a tag to the newly created Vp2 parameter.
If it is unable to create Vp2, then it will return -1.

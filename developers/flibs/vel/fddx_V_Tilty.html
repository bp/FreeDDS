fddx_V_Tilty - return a tag to a Tilty parameter.

SYNOPSIS

#include <fddx_vel.h>

integer fddx_V_Tilty(vel_tag)

integer vel_tag

DESCRIPTION

fddx_V_Tilty returns a tag to the Tilty parameter of a velocity.
If the velocity does not have Tilty defined, then fddx_V_Tilty returns -1.

INPUT

The parameter, "vel_tag", specifies a velocity for which to find the Tilty parameter.

RETURN VALUE

fddx_V_Tilty returns a tag to the Tilty parameter.
If Tilty is not defined, then it will return -1.

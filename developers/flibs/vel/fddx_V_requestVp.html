fddx_V_requestVp - read a Vp parameter.

SYNOPSIS

#include <fddx_vel.h>

integer fddx_V_requestVp(vel_tag, critical)

integer vel_tag
integer critical

DESCRIPTION

fddx_V_requestVp reads a Vp parameter.
The keyword prefix for the users is "vp". See also function fddx_V_newElement.

INPUT

The parameter, "vel_tag", specifies a velocity with which to associate this Vp parameter.

The parameter, "critical", specifies whether this parameter is critical.
If "critical" is non-zero and fddx_V_requestVp is unable to find a suitable Vp parameter, then a DDS error is printed.
NOTE that in the event of an error the program continues in order to allow for clean stopping of MPI codes.

RETURN VALUE

fddx_V_requestVp returns a tag to the newly created Vp parameter.
If it is unable to create Vp, then it will return -1.

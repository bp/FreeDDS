fddx_V_Vs - return a tag to a Vs parameter.

SYNOPSIS

#include <fddx_vel.h>

integer fddx_V_Vs(vel_tag)

integer vel_tag

DESCRIPTION

fddx_V_Vs returns a tag to the Vs parameter of a velocity.
If the velocity does not have Vs defined, then fddx_V_Vs returns -1.

INPUT

The parameter, "vel_tag", specifies a velocity for which to find the Vs parameter.

RETURN VALUE

fddx_V_Vs returns a tag to the Vs parameter.
If Vs is not defined, then it will return -1.

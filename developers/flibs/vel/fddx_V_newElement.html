fddx_V_newElement - read a new velocity parameter.

SYNOPSIS

#include <fddx_vel.h>

integer fddx_V_newElement(alias)

character*(*)alias

DESCRIPTION

fddx_V_newElement reads a new velocity Element.
The type of element can be defined by the user.
Currently 3D grids and single values are implemented.
For a 3D grid, fddx_V_newElement scans for a keyword defined by "alias" and the velocity from a file.
For a constant value, fddx_V_newElement scans for a keyword defined by "alias_single" followed by a value.
If both "alias" and "alias_single" are specified, then fddx_V_newElement returns an error.

INPUT

The parameter, "alias", specifies the keyword prefix to use. (see description above).

RETURN VALUE

fddx_V_newElement returns a tag to the newly created velocity parameter. In the event of an error, it will return -1.

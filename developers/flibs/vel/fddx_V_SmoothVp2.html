fddx_V_SmoothVp2 - return a tag to a SmoothVp2 parameter.

SYNOPSIS

#include <fddx_vel.h>

integer fddx_V_SmoothVp2(vel_tag)

integer vel_tag

DESCRIPTION

fddx_V_SmoothVp2 returns a tag to the SmoothVp2 parameter of a velocity.
If the velocity does not have SmoothVp2 defined, then fddx_V_SmoothVp2 returns -1.

INPUT

The parameter, "vel_tag", specifies a velocity for which to find the SmoothVp2 parameter.

RETURN VALUE

fddx_V_SmoothVp2 returns a tag to the SmoothVp2 parameter.
If SmoothVp2 is not defined, then it will return -1.

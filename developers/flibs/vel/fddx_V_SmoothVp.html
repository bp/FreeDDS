fddx_V_SmoothVp - return a tag to a SmoothVp parameter.

SYNOPSIS

#include <fddx_vel.h>

integer fddx_V_SmoothVp(vel_tag)

integer vel_tag

DESCRIPTION

fddx_V_SmoothVp returns a tag to the SmoothVp parameter of a velocity.
If the velocity does not have SmoothVp defined, then fddx_V_SmoothVp returns -1.

INPUT

The parameter, "vel_tag", specifies a velocity for which to find the SmoothVp parameter.

RETURN VALUE

fddx_V_SmoothVp returns a tag to the SmoothVp parameter.
If SmoothVp is not defined, then it will return -1.

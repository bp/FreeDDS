fddx_V_requestVs - read a Vs parameter.

SYNOPSIS

#include <fddx_vel.h>

integer fddx_V_requestVs(vel_tag, critical)

integer vel_tag
integer critical

DESCRIPTION

fddx_V_requestVs reads a Vs parameter.

INPUT

The parameter, "vel_tag", specifies a velocity with which to associate this Vs parameter.
The keyword prefix for the users is "vs". See also function fddx_V_newElement.

The parameter, "critical", specifies whether this parameter is critical.
If "critical" is non-zero and fddx_V_requestVs is unable to find a suitable Vs parameter, then a DDS error is printed.
NOTE that in the event of an error the program continues in order to allow for clean stopping of MPI codes.

RETURN VALUE

fddx_V_requestVs returns a tag to the newly created Vs parameter.
If it is unable to create Vs, then it will return -1.

fddx_V_requestTiltx - read a Tiltx parameter.

SYNOPSIS

#include <fddx_vel.h>

integer fddx_V_requestTiltx(vel_tag, critical)

integer vel_tag
integer critical

DESCRIPTION

fddx_V_requestTiltx reads a Tiltx parameter. This is is the tilt angle in the x-direction for TTI velocities.

INPUT

The parameter, "vel_tag", specifies a velocity with which to associate this Tiltx parameter.
The keyword prefix for the users is "tiltx". See also function fddx_V_newElement.

The parameter, "critical", specifies whether this parameter is critical.
If "critical" is non-zero and fddx_V_requestTiltx is unable to find a suitable Tiltx parameter, then a DDS error is printed.
NOTE that in the event of an error the program continues in order to allow for clean stopping of MPI codes.

RETURN VALUE

fddx_V_requestTiltx returns a tag to the newly created Tiltx parameter.
If it is unable to create Tiltx, then it will return -1.

fddx_V_Vp - return a tag to a Vp parameter.

SYNOPSIS

#include <fddx_vel.h>

integer fddx_V_Vp(vel_tag)

integer vel_tag

DESCRIPTION

fddx_V_Vp returns a tag to the Vp parameter of a velocity.
If the velocity does not have Vp defined, then fddx_V_Vp returns -1.

INPUT

The parameter, "vel_tag", specifies a velocity for which to find the Vp parameter.

RETURN VALUE

fddx_V_Vp returns a tag to the Vp parameter.
If Vp is not defined, then it will return -1.

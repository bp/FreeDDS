fddx_V_show - print a summary of a velocity.

SYNOPSIS

#include <fddx_vel.h>

integer fddx_V_show(vel_tag)

integer vel_tag

DESCRIPTION

fddx_V_show prints a summary of a velocity to the printout file.
It lists which parameters are defined, and summary information about each one.

INPUT

The parameter, "vel_tag", specifies which velocity to show.

RETURN VALUE

fddx_V_show always returns 0.

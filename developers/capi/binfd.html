cdds_binfd - return the file descriptor (fd) associated with a binary file

SYNOPSIS

#include "cdds.h"

int cdds_binfd(bin_tag);

BIN_TAG bin_tag

DESCRIPTION

cdds_binfd returns the file descriptor (fd) of the binary file. A value of -1 is returned on any error.

INPUT

bin_tag is a binary tag, previously obtained from cdds_open. It specifies a particular binary stream.

EXAMPLE

BIN_TAG bin;
int fd;
...
fd = cdds_binfd(bin);
...

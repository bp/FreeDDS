cdds_calloc - memory allocator

SYNOPSIS

#include "cdds.h"

void *cdds_calloc(nobj, size);

size_t nobj
size_t size

DESCRIPTION

This function allocates memory. If zero bytes are requested, then a NULL pointer is returned. The DDS memory routines allow for runtime memory checking. See standard man pages for calloc(3).

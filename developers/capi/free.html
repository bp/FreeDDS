cdds_free - free allocate memory

SYNOPSIS

#include "cdds.h"

void cdds_free(ptr);

void * ptr

DESCRIPTION

This function frees memory. If a NULL pointer is passed, then no freeing of memory is done. The DDS memory routines allow for runtime memory checking. See standard man pages for free(3).

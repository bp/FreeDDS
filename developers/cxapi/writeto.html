cddx_writeto - seek and write to a DDS dataset

SYNOPSIS

#include "cdds.h"

int function cddx_writeto(bin_tag, offset, buffer, count);

BIN_TAG bin_tag
long offset
void *buffer
int count

DESCRIPTION

Routine, "cddx_writeto", writes "count" traces to the output dataset referenced by "bin_tag" starting from the offset trace location number, "offset", form array "buffer". An "offset" value of "0" will write starting at the first trace in the dataset. The number of traces written is returned or < 0 on error.

cddx_addaxis - add a new axis to a DDS dictionary

SYNOPSIS

#include "cdds.h"

int function cddx_addaxis(bin_tag, num, name);

BIN_TAG bin_tag
int num
char *name

DESCRIPTION

Routine, "cddx_addaxis", adds a new axis to an output dataset prior to the final opening. The axis number, "num", (should be between 1 and the number of current axes + 1) specifies the axis number for the newly created axis with "name". There is no check made on the "name" to assure that it does not already exist. A value of 0 is normally returned; < 0 on error.

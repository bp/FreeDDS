cddx_getbin - retrieve external DDS binary tag

SYNOPSIS

#include "cdds.h"

BIN_TAG function cddx_getbin(bin_tag);

BIN_TAG bin_tag

DESCRIPTION

Routine, "cddx_getbin", returns the BIN_TAG of the external binary stream. This is needed because these convenience routines only use the internal buffer BIN_TAG's. A return value < 0 indicates an error.

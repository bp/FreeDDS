cfft_nrfft5 - return mixed radix 2,3,5 size

SYNOPSIS

#include <cfft.h>

int cfft_nrfft5(n);

int n

DESCRIPTION

The function, "cfft_nrfft5", is provided for determining an appropriate radix 2,3,5 value to pad out to for the FFT's. Given the integer input, "n", the function returns the integer M such that M is the smallest number greater than or equeal to "n" such that M = 2**i * 3**j * 5**k where i is a positive integer and j & k are non-negative integers.

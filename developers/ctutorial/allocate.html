Dynamic memory allocation can be done in "C" with the use of the DDS Memory Allocation routines. By using the DDS Memory Allocation routines, any DDS program can check memory overwrites at run time by the use of the command line or environmental varable, DDS_DEBUG, setting it to include "dgb_call" (see Debug Support). A couple of examples are: 

      databuf = cdds_malloc(ns*sizeof(float));
c
      nbytes = cdds_prec(in_bin, 0);
      inbuf = cdds_malloc(nbytes);

Whenever dynamic memory needs to be released, only use the DDS free routine to free the memory with the array pointer name: 

      cdds_free(inbuf);

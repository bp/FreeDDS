Open Internal Output Buffer

Occassionally, a different output buffer must be used separately from the input buffer because something like the output size of the first axis or the SAMPLE_TYPE may have changed. So a new internal buffer can be opened and modified. For example if you need to change the output to complex Samples and change the number of Samples: 

outbuf_dict = cdds_out("", "", buf_dict);
cdds_printf("fmt:SAMPLE_TYPE",
          "typedef complex SAMPLE_TYPE;\n");
cdds_printf("size.axis(1)", "%d\n", nout);
            
/* Open "outbuf" binary */
outbuf_bin = cdds_open(outbuf_dict, " ", " ", "m");

Open External Output Dataset

Open the output dataset from the internal buffer by: 

out_dict = cdds_out("out", "stdout:", outbuf_dict);
out_bin = cdds_open(out_dict, "out_format", "out_data", "w+");

If a file already exists and it just needs to be modified then use cdds_mod. If the dictionary already exists, the title and "par:" dictionaries are appended to it (like cdds_out). If the dictionary must be created, the history provided by the fifth argument is used to initialize it (like cdds_in). If a dictionary is not specified by the user or default, a temporary dictionary is created and a warning is issued. The resulting dictionary is left in "print" mode. 

out_dict = cdds_mod("out", "", title, outbuf_dict);
out_bin = cdds_open(out_dict, "out_format", "out_data", "r+");

Open Output From Scratch

If you need to generate an output dataset with no input dataset, then you will need to create a dummy input dictionary first with: 

dmy_dict = cdds_in("", "", title);

and then use this dummy input dictionary, in_dict, when opening any output dictionaries. You will need to define the actual format for example by: 

outbuf_dict = cdds_out("", "", dmy_dict);
cdds_printf("fmt:ASP_TRACE", "typedef struct {\n");
cdds_printf("", "integer StaCor;\n");
cdds_printf("", float SrcX;\n");
cdds_printf("", SAMPLE_TYPE Samples[axis_size(1)];\n");
cdds_printf("", } ASP_TRACE;\n");
  
cdds_printf("fmt:SAMPLE_TYPE", 
          "typedef float SAMPLE_TYPE;\n");

cdds_printf("axis", "t x y\n");
cdds_printf("size.t", "%d\n", nt);
cdds_printf("size.x", "%d\n", nx);
cdds_printf("size.y", "%d\n", ny);
cdds_printf("delta.t", "%f\n", si);
cdds_printf("format", "asp\n");
            
/* Open "outbuf" binary */
outbuf_bin = cdds_open(outbuf_dict, "", "", "m");

fddx_wtime - wall-time

SYNOPSIS

#include <fdds.h> (fixed-format sources)
#include <f90dds.h> (free-format sources)

real*8 function fddx_wtime()

DESCRIPTION

Routine, "fddx_wtime", returns wall-clock time in seconds from the first time this routine gets called.

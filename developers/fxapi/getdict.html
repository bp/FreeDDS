fddx_getdict - retrieve the DDS dictionary name

SYNOPSIS

#include <fdds.h> (fixed-format sources)
#include <f90dds.h> (free-format sources)

integer function fddx_getdict(dict, bin_tag)

character(*) dict
integer bin_tag

DESCRIPTION

Routine, "fddx_getdict", returns the internal buffer Dictionary associated with the given bin_tag. If the binary is opened for output, then the output Dictionary is returned. This provides a way of modifying a dictionary after the binary has been written and closed. A return value < 0 indicates an error; otherwise a 0 is returned.

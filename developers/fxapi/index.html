fddx_index - retrieve index to a DDS field

SYNOPSIS

#include <fdds.h> (fixed-format sources)
#include <f90dds.h> (free-format sources)

integer function fddx_index(bin_tag, name, prime)

integer bin_tag
character(*) name
integer prime

DESCRIPTION

Routine, "fddx_index", returns an offset index into the trace array for the trace field defined by "name" with data type "prime". The return value will be -1 on an error. The specified "prime" type can be any prime type defined for "fdds_index".

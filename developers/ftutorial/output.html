Open Internal Output Buffer

Occassionally, a different output buffer must be used separately from the input buffer because something like the output size of the first axis or the SAMPLE_TYPE may have changed. So a new internal buffer can be opened and modified. For example if you need to change the output to complex Samples and change the number of Samples: 

      ier=fdds_out(outbuf_dict,' ',' ',buf_dict)
      ier=fdds_printf('fmt:SAMPLE_TYPE',
     *    'typedef complex SAMPLE_TYPE;\n\0')
      ier=fdds_printf('size.axis(1)','%d\n\0',nout)
            
c     Open "outbuf" binary
      outbuf_bin=fdds_open(outbuf_dict,' ',' ','m')

Open External Output Dataset

Open the output dataset from the internal buffer by: 

      ier=fdds_out(out_dict,'out','stdout:',outbuf_dict)
      out_bin=fdds_open(out_dict,'out_format','out_data','w+')

If a file already exists and it just needs to be modified then use fdds_mod. If the dictionary already exists, the title and "par:" dictionaries are appended to it (like fdds_out). If the dictionary must be created, the history provided by the fifth argument is used to initialize it (like fdds_in). If a dictionary is not specified by the user or default, a temporary dictionary is created and a warning is issued. The resulting dictionary is left in "print" mode. 

      ier=fdds_mod(out_dict,'out',' ',title,outbuf_dict)
      out_bin=fdds_open(out_dict,'out_format','out_data','r+')

Open Output From Scratch

If you need to generate an output dataset with no input dataset, then you will need to create a dummy input dictionary first with: 

      ier=fdds_in(dmy_dict,' ',' ',title)

and then use this dummy input dictionary, dmy_dict, when opening any output dictionaries. You will need to define the actual format for example by: 

      ier=fdds_out(outbuf_dict,' ',' ',dmy_dict)
      ier=fdds_printf('fmt:ASP_TRACE','typedef struct {\n\0')
      ier=fdds_printf(' ','int StaCor;\n\0')
      ier=fdds_printf(' ',float SrcX;\n\0')
      ier=fdds_printf(' ',SAMPLE_TYPE Samples[axis_size(1)];\n\0')
      ier=fdds_printf(' ',} ASP_TRACE;\n\0')
  
      ier=fdds_printf('fmt:SAMPLE_TYPE',
     *    'typedef float SAMPLE_TYPE;\n\0')

      ier=fdds_printf('axis','t x y\n\0')
      ier=fdds_printf('size.t','%d\n\0',nt)
      ier=fdds_printf('size.x','%d\n\0',nx)
      ier=fdds_printf('size.y','%d\n\0',ny)
      ier=fdds_printf('delta.t','%f\n\0',si)
      ier=fdds_printf('format','asp\n\0')
            
c     Open "outbuf" binary
      outbuf_bin=fdds_open(outbuf_dict,' ',' ','m')

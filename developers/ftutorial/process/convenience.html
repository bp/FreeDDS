Binary Read

There are three Fortran Convenience routines for reading data. These routines work in conjunction with the binary tags returned from the Convenience open routines and simplify the process by hiding the extra internal binaries for the required double buffering that takes place along with the mapping.

fddx_read is called to read one or more input traces from a dataset.

fddx_readfrom does a seek first to a specified trace position and then reads the traces.

fddx_readall does it all. It opens and reads an entire dataseta (e.g. velocity volume).

For example: 

      integer in_bin
      real buf(*)
      ...
c     read 10 traces into buffer
      ier=fddx_read(in_bin,buf,10)

Binary Write

There are two Fortran Convenience routines for writing data. These routines work in conjunction with the binary tags returned from the Convenience open routines and simplify the process by hiding the extra internal binaries for the required double buffering that takes place along with the mapping.

fddx_write writes one or more traces to an output dataset.

fddx_writeto does a seek first to a specified trace position and then writes the traces.

For example: 

      integer out_bin,n
      real buf(*)
      ...
c     write 3 traces starting at the n'th trace
      ier=fddx_writeto(out_bin,n-1,buf,3)

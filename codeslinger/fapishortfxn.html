There are relatively few DDS functions that are required for typical application development. These few functions can be adapted to 95% of application needs. All the other functions are available for special needs as they arise, but if you only learn functions on the short list, you should be able to perform almost any task and greatly simplfy your coding experience.

Sort by Utility

    Open/Close, Binary Control, Dictionary Access
    • fddx_in - open input (drop headers, pass samples)
    • fddx_in2 - open input (pass headers andsamples)
    • fddx_out - open output
    • fdds_close - close dictionary and associated binary

    • fddx_read - read data
    • fddx_write - write data
    • fdds_lseek8 - trace seek

    • fdds_dict - set dictionary mode
    • fddx_dict - set dictionary mode
    Print File Routines
    • fdds_openpr - open print file
    • fdds_closepr - close print file , exit program
    • fdds_prtcon - print to print file and console
    • fdds_prtmsg - print to print file
    • fdds_prterr - print error to console and print file
    Utility Routines
    • fdds_axis_prod - product of axis sizes
    • fdds_errors - return # errors
    • fdds_free - free memory
    • fdds_getxx - get a field
    • fdds_index - query field index
    • fdds_malloc8 - allocate memory
    • fdds_member - query member tag
    • fdds_printf - print definitions
    • fdds_putxx - put a field
    • fdds_scanf - scan definitions
    • fdds_scank - scan definitions for keys
    Parallel setup Utilities
    • fdds_initmpi - initialize MPI
    • fdds_initopenmp - initialize OpenMP

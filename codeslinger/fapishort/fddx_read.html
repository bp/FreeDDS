fddx_read - read a DDS dataset

SYNOPSIS

#include <fdds.h> (fixed-format sources)
#include <f90dds.h> (free-format sources)

integer function fddx_read(bin_tag, buffer, count)

integer bin_tag
real buffer(*)
integer count

DESCRIPTION

fddx_read reads count traces from the input binary stream referenced by bin_tag into array buffer.

OUTPUT

fddx_read returns the number of traces read. If an error occurs, the return value is negative.

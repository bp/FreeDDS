fdds_errors - query errors

SYNOPSIS

#include <fdds.h> (fixed-format sources)
#include <f90dds.h> (free-format sources)

integer function fdds_errors()

DESCRIPTION

fdds_errors returns the number of errors detected, as defined by the number of times fdds_prterr has been used.

EXAMPLES

      integer ier,pct
      ...
      if (fdds_errors().eq.0) then
         ier=fdds_prtcon('%3d% Complete\n\0',pct)
      endif

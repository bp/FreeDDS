Purpose:   Retrieve controls from the command line and parfile and pass processed data to an output file. Examlpe of 2D i/o.

Code Outline:

  • Open a file for access to samples
  • Retrieve data characteristics needed for the task (n1, n2, ..., d1, d2, ...)
  • Retrieve parameters from the user
  • Open the output
  • Allocate an array to hold data
  • Read the data, apply the scalar and bias
  • Write the output
  • Clean up before exiting

New DDS resources:

  • fdds_dict / cdds_dict - set dictionary mode
  • fddx_out / cddx_out - open output
  • fdds_lseek8 / cdds_lseek8 - trace seek
  • fddx_write / cddx_write - write data

Purpose:   Make use of DDS print files. Write information to the terminal and to print files. Demonstrate interfaces between main program and functions or subroutines.

Code Outline:

  • Open the print file, detect request for help message
  • Open a file for access to samples
  • Retrieve data characteristics needed for the task (n1, n2, ..., d1, d2, ...)
  • Use a subroutine or function to retrieve user parameters
  • Open the output
  • Allocate an array to hold data and internal vectors
  • Use a subroutine or function to build the gain curve
  • Read the data, apply the gain, periodically report progress
  • Write the output
  • Clean up and exit

New DDS resources:

  • fdds_openpr / cdds_openpr - open print file
  • fdds_prtcon / cdds_prtcon - print message to console and print file
  • fdds_prtmsg / cdds_prtmsg - print message to print file if open (else console)
  • fdds_prterr / cdds_prterr - print error to console and print file
  • fdds_closepr / cdds_closepr - close print file and exit

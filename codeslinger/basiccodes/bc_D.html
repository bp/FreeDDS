Purpose:   Learn to modify output dictionary when an algorithm warrants.

Code Outline:

  • Open the print file, detect request for help message
  • Open a file for access to samples
  • Retrieve data characteristics needed for the task (n1, n2, ..., d1, d2, ...)
  • Retrieve user parameters
  • Open the output dictionary
  • Print corrected definitions to output dictionary
  • Open the output file
  • Allocate arrays to hold input and output data
  • Read the data, apply the algorithm, periodically report progress
  • Write the output
  • Clean up and exit

New DDS resources:

  • fdds_printf / cdds_printf - print definitions
  • fddx_dict / cddx_dict - set dictionary mode

Purpose:   To demonstrate methods of changing trace headers.

Outline:

  • Retrieve users mapping overrides
  • Define new trace structure and new header mapping in override: dictionary
  • Open a data file for access to headers and samples
  • Retrieve data characteristics needed for the task (n1,n2, ...)
  • Open output dictionary
  • Print old_format=asp to output dictionary to force new headers in output
  • Open the output file
  • Allocate an array to hold data
  • Read data
  • Write the output
  • Clean up before exiting

Purpose:   Use FFT wrappers to change REAL samples to COMPLEX. Adjust output dictionary to reflect new sample type.

Code Outline:

  • Open the print file, detect request for help message
  • Open a file for access to samples
  • Retrieve data characteristics needed for the task (n1, n2, ..., d1, d2, ...)
  • Retrieve user parameters
  • Set internal data sizes and deltas
  • Open the output dictionary
  • Print new definitions to output dictionary
  • Open the output file
  • Allocate data array
  • Read the data, transform, periodically report progress
  • Write the output
  • Clean up and exit

New DDS resources:

  • fft_nrfft5 / cfft_nrfft5 - return mixed-radix 2,3,5, size
  • fft_rcfftm / cfft_rcfftm - real-to-complex FFT of many traces

Basic program flow for a common DDS application
(This color indicates use of DDS API)




Initialize Title
      information transfered to output dictionary

Open the print file
      the print file is for logging I/O and runtime information

Open the input

Retrieve data characteristics
      sizes, origins, deltas of all axes

Set defaults for and retrieve other parameters
      some may be from input dictionary, others from the user.

Open the output
      only an ascii dictionary until the first seek or binary i/o

Allocate arrays

Set up the algorithm
      precompute constant scalars, vectors, etc. needed by the algorithm

Loop over input data
      read
      apply algorithm
      write
End Loop

Clean up
      free allocated memory, close files

End

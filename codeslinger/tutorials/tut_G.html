Purpose:   Access headers for information or use in an algorithm.

Task:  

  1. Find the ranges of user selectable trace headers.
  2. Multiply data samples by user selectable trace headers.

Outline:

  • Open a file for access to headers
  • Retrieve user overrides for headers to analyze
  • Retrieve data characteristics needed for task (n1, n2, ..., header word locations)
  • Allocate an array to hold data
  • Read through the data, retrieve trace headers for use by algorithm
  • Clean up before exiting

New DDS resources:

  • fdds_member / cdds_member - query member tag
  • fdds_index / cdds_index - query field index
  • fddx_getxx / cddx_getxx - get a field

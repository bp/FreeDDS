Purpose:   Demonstrate the use of OpenMP in a DDS application

Task:  

Warning:   DDS is not thread safe. Never, never, use DDS in parallel regions.

Outline:

  • Open a data file for access to samples
  • Retrieve data characteristics needed for the task (n1,n2, ...)
  • Open a geometry file for access to samples
  • Retrieve geometry sizes
  • QC conformance between data and geometry, report discrepancies
  • Open output
  • Allocate an arrays to hold data and geometry
  • Read through the data and geometry
  • Use OpenMP to process the data based on geometry
  • Write the output
  • Clean up before exiting

New DDS resources:

  • fddx_initopenmp / cddx_initopenmp - initialize OpenMP

Purpose:   To demonstrate how multiple inputs are coordinated.

Task:   Write a DDS program that reads data from one file and geometry from another. Implement a process that uses geometry to control the function applied.

Outline:

  • Open a data file for access to samples
  • Retrieve data characteristics needed for the task (n1,n2, ...)
  • Open a geometry file for access to samples
  • Retrieve geometry sizes
  • QC conformance between data and geometry, report discrepancies
  • Open output
  • Allocate arrays to hold data and geometry
  • Read through the data and geometry
  • Process the data based on geometry
  • Write the output
  • Clean up before exiting

These are various example codes that may or may not prove useful.

Where noted, codes employ OpenMP and/or MPI.

  1. zxy_extract.F
  2. zxy_extract.c
    • Extract user specified z,x,y range from an input volume.
    • Users specify ranges with origins and widths in input data coordinates
    • Where range is within input volume, just truncate samples
    • Where range is outside input volume, pad by replication
    • Pass trace headers if any
    • 2D reads, 3D write

  3. mute.F
    • Apply a mute based on user t0_mute,v_mute and header XY values
    • Use source/receiver XY headers as defaults, allow user overrides
    • An example of reasonably defensive and robust coding.

  4. lowpass.F
    • Apply a 3D low-pass fiter to 3D volumes
    • Real-to-complex and complex-to-real 3D FFTs
    • OpenMP in filter application and FFTs
    • Preserve top mute, preserve trace headers when present
    • Force internal sample alignment to complex word boundaries

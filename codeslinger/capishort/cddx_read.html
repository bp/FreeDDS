cddx_read - read a DDS dataset

SYNOPSIS

#include "cdds.h"

int function cddx_read(bin_tag, buffer, count);

BIN_TAG bin_tag
void *buffer
int count

DESCRIPTION

cddx_read reads count traces from the input binary stream referenced by bin_tag into array buffer.

OUTPUT

cddx_read returns the number of traces read. If an error occurs, the return value is negative.

cdds_malloc - allocate memory

SYNOPSIS

#include "cdds.h"

void *cdds_malloc(size);

size_t size

DESCRIPTION

This function allocates memory. If zero bytes are requested, then a NULL pointer is returned. The DDS memory routines allow for run time memory checking. See standard man pages for malloc(3).

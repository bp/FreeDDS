cdds_scant - find a definition in a dictionary and retrieve one token

SYNOPSIS

#include "cdds.h"

int cdds_scant(alias, str);

const char *alias
const char **str

[ Description | Arguments | Output | Examples ]

DESCRIPTION

cdds_scant retrieves one token from the value of a definition in a dictionary. The "scan" dictionary(s) must have been previously selected by cdds_dict, cddx_dict, or cddx_out.

ARGUMENT

"alias" is used to set the "position" within the "scan" dictionaries from which the token is retrieved. It specifies the definition that contains the token. alias may contain multiple names, delimited by white space, or it may be null. When names are given, the "scan" dictionaries are searched from bottom-to-top, right-to-left. The newest definition that matches any name is selected and the "position" is set immediately after the "=" of the definition. If a null alias is given (i.e. NULL, zero length string, or a string that only contains white space) the next token in the current value is processed.

"*str" is a string pointer that is assigned by cdds_scant. The string contains the next token of the current value. It uses a temporary buffer that can be changed by the next call to the DDS API.

OUTPUT

cdds_scant returns the number of characters in the token assigned to *str (not including the NULL terminator "\0") and the "position" is advanced to EOT (end of token). If a token is not found, cdds_scant return EOF and *str is not changed.

EXAMPLES

   /* Retrieve a user override for SrcX */
   int ier;
   char *SrcX;
   const char *defn;

   cdds_dict("par:","scan");

   ier=cdds_scant("SrcX",&defn);
   if (ier > 0 ) {
      SrcX=(char *)malloc(ier+1);
      strcpy(SrcX,defn);
   } else {
      SrcX="SrcX";
   }


   /* Load an array of strings with axis names */
   int ier, nhdr, ihdr;
   BIN_TAG fd_in;
   char *axes[RANK_MAX], *Title;
   const char *defn;

   fd_in=cddx_in("in","stdin:",Title);

   nhdr=cdds_scank("axis","");
   for ( ihdr=0, ihdr<nhdr; ihdr++ ) {
      ier=cdds_scant("",&defn);
      axes[ihdr]=(char *)malloc(ier+1);
      strcpy(axes[ihdr],defn);
   }

cdds_free - free allocate memory

SYNOPSIS

#include "cdds.h"

void cdds_free(ptr);

void *ptr

DESCRIPTION

cdds_free frees allocated memory. If a NULL pointer is passed, then no memory is freed. The DDS memory routines allow run time memory checking. See standard man page for free(3).

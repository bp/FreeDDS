cdds_initopenmp - initialize the DDS system for OpenMP parallelization

SYNOPSIS

#include "cdds.h"

int cdds_initopenmp();

DESCRIPTION

cdds_initopenmp is used to initialize the DDS system for OpenMP parallelization. If OpenMP is available, it scans the command line for "nproc" and calls "omp_set_num_threads" to set the number of processors to use. The default value for "nproc" is 1. The return value is "nproc". If OpenMP is not available, it simply returns 1.

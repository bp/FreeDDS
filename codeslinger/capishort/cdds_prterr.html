cdds_ptrerr - write an error message

SYNOPSIS

#include "cdds.h"

int cdds_prterr(fmt, ...);

const char *fmt

[ Description | Examples ]

DESCRIPTION

cdds_prterr writes an error message to the console. The program name will be indicated on each line of the message. If the print file has been opened, the message will also be written to the print file. The message is based on the format string, fmt, and optional arguments (see cdds_printf). An error message will cause cdds_closepr to print an abnormal termination message. The return value is always 0.

EXAMPLES


   cdds_prterr("illegal value=%f\n", value);

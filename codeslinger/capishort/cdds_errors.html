cdds_errors - query the number of print error messages

SYNOPSIS

#include "cdds.h"

int cdds_errors();

[ Description | Examples ]

DESCRIPTION

cdds_errors returns the number of errors detected, as defined by the number of times cdds_prterr has been used.

EXAMPLES


   int ier;

   ier = cdds_errors();
   if (ier) {
      cdds_prtcon("%d error reported\n", ier);
   }

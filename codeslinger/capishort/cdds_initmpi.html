cdds_initmpi - initialize the DDS system for MPI parallelization

SYNOPSIS

#include "cdds.h"
#include "mpi.h"

int cdds_initmpi(node, nodes);

int *node
int *nodes

DESCRIPTION

cdds_initmpi is used to initialize the DDS system for MPI parallelization. It determines the number of MPI nodes used and the node number (0 through nodes-1) for the current process. If MPI is available on the current system, MPI is initialized with MPI_Init. The master node then passes all the command line parameters to the slave nodes to ensure all nodes "see" the exact same parameters (since MPI does not guarantee that). If MPI is not available, it simply checks that "np=1". The return value is alway 0.

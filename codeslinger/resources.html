Other DDS Resources

Since no project is completely straight forward, the following resources are sometimes useful in solving problems with 'make', understanding variables set by DDS, and getting the true story behind what DDS functions really do.

Manipulating 'make'

  • $DDSROOT/src/lib/host/HostDir -   List architecture-dependent paths for include files, libraries, objects, and executables.

  • $DDSROOT/src/lib/host/HostComp -   Reports compiler information, compiler flags, linker, and linker flags for the target architecture.

  • $DDSROOT/src/lib/host/HostMP -   Reports OpenMP and MPI flags for compiler and linker, shows the libraries used, paths to those libraries and the mpirun executable.

  • $DDSROOT/etc/MakeVariables -   Sets many variables used during 'make'. Users are able to replace or modify these as warranted... add/remove compile flags, libraries, include paths.

  • $DDSROOT/etc/MakeRules -   Sets rules for various targets. Useful as a go-by if boutique rules are needed.

Source Code Variables set by DDS

  • $DDSROOT/include/${TArchBinDir}${CompVer}/fdds.h
       Architecture-dependent #include file for F77

  • $DDSROOT/include/${TArchBinDir}${CompVer}/f90dds.h
       Architecture-dependent #include file for F90

  • $DDSROOT/include/cdds.h
       #include file for C

  • $DDSROOT/include/fft.h
       F77 #include file fft wrappers

  • $DDSROOT/include/f90fft.h
       F90 #include file fft wrappers

  • $DDSROOT/include/cfft.h
       C #include file fft wrappers

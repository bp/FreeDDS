Entering the "Dump Samples" command ("DS <#>") will printout trace Sample values for each trace in record #. If no # is given, then it will printout the Sample values for the next record in sequence. Once this is done, then each time a blank command is entered it will printout the Sample values from the next record or if you enter just a # by itself, it will printout the Sample values for the specified record. You can use the "SS" command to select which Samples to printout and "ST" command to select which traces to printout. A summary of the minimum, maximum and RMS values are pinted out at the end of each dump. The example shown below was setup with "ss 45 100 2" and "st 5 15 2".

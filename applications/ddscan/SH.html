Entering the "Select Trace Headers" command ("SH") will select the trace headers to printout when using commands "DH" or "AH". To use this command, enter "SH" followed by one or more trace header names (see the "T" command), one at a time on separate lines, ending by entering a blank line. See the example below. 

sh
  
 Enter Header Names (end with blank):
DstSgn
StaCor

  
 Enter command:
dh

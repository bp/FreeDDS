Entering the "Dump Headers" command ("DH <#>") will printout selected trace headers (see the "SH" command) for each trace in record #. If no # is given, then it will printout the trace headers for the next record in sequence. Once this is done, then each time a blank command is entered it will printout the headers from the next record or if you enter just a # by itself, it will printout the headers from the specified record. You can use the "ST" command to select the traces to have printed out. The example shown below used "st 1 10" to select the traces to dump. 

dh 24

After opening a dataset, XSD takes the dataset axis information and sets up the data definition portion of the "Server Parameters" Window. You can override any of these parameters (with the exception of the size of the first axis) including changing the "Axis Rank" (# of dimensions). Change any of the "Axis" names to appear on the plots. Change the "Delta', "Origin', "Base" and "Step" to allow selecting and/or displaying relative to them. For example time axis is always setup in seconds from DDS. If you wish to display time in msec, then just change the corresponding "Delta" value (units are not displayed). Change the "Axis Rank" and/or Axis "Sizes", but beware that selecting parameters beyond the dataset will prevent the ddsServer from being able to read the data.

You can select data definition parameters from any currently displayed window by selecting the window's Id # from the "Data Definition from:" option list. Alternatively, you can press the "Reset Definition" button to reset the definition parameters from the dataset.

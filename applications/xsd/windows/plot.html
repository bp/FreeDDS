From any Display Window select "Window→Plot/Save Image" to pop-up the "Print Parameters Window". (It will be different for Animation Windows).

  • From Single Display Windows:
    Select this button to send the image to a printer and/or create a gif or tiff file of the image. The "Print Parameters Window" will pop-up. Set the following fields then press the "OK" button to submit the request or "Cancel" to simply close the window.
    • "Plot Title" - text to add at the top of the output image
    • "Horizontal Title" - text to add along the horizontal axis
    • "Vertical Title" - text to add along the vertical axis
    • "Border in pixels" - size of the border around the image
    • "Output file" - specify the name of the image file to create for the "gif" or "tiff" file. If left blank, no file is created. A suffix of ".gif" or ".tiff" will be added if not specified.
    • "Note file" - name of file containing "notes" (in the form of an ascii text file) to be added to the output image
    • "Printer Name" - specify the name of a printer which to print the file. If left blank, no file it printed.
    • "Scale (DPI)" - the scale factor in "Dots Per Inch" which is used when creating "tiff" files
    • "GIF Format" - specifies the output to be a "gif" format file
    • "TIFF Format" - specifies the output to be a "tiff" format file
    • "Color Scale" - specifies whether to include a color Scale on the image
    • "Save current settings" - specifies saving the current settings for the Print Parameters Window

  • From Animation Display Windows:
    If "Window→Plot/Save Image" is selected from an Animation Window then the "Print Parameters Window" will be modified. The output file will be created as an animated "gif" file.
    • "Plot Title" - text to add at the top of the output image. This can include special format characters to allow the title to vary with each animation image.
      • "%[#]g#" - this is the way to include the axis value from any axis for each animation image. The first "#" optionally specifies the length of the field. The second "#" specifies the number of decimal digits to display. For example if each animation image is a time slice for different times, you can use "%6g3" to display values in a format like " 3.025".
      • "%s" - this is the way to include the file base name for each image. This is useful when the various animation images were taken from different files.
    • "Horizontal Title" - text to add along the horizontal axis
    • "Vertical Title" - text to add along the vertical axis
    • "Delays (msec)" - the time to delay between each image. Multiple values can be specified with each one being applied the each corresponding animation image. If there are fewer Delays than images, then the last value will be used for the remaining images. If there are more Delays than images, then the last Delay values will not be used.
    • "NumLoops" - specify the number of times to loop through all the images. A value of "0" will cause the animated "gif" to loop indefinitely.
    • "Output gif file/base" - specify the name of the "gif" file. A suffix of ".gif" will be added if not specified.
    • "Display Axis" - specifies annotating the axes
    • "Save individual files" - specifies saving each animated image to separate files
    • "Save current settings" - specifies saving the current settings for the Print Parameters Window

The Animation Window is a composite of multiple Display Windows which can animate rapidly through all the data images. The only difference in the controls is the animation control area just below the menu panel. The "Play" button will animate through the images with the delay between each image as specified by the "Movie Delay (ms)" value. The images can be stepped through one at a time either forward ("Step->") or reverse ("<-Step") direction. The animation window is deleted by either clicking on the "Quit" button or selecting "Window→Delete Window" from the menu panel.

The other controls behave pretty much the same way as with the basic Display Windows. The act upon the corresponding Display image currently displayed. Anything done to the image will also act upon the original Display Window.

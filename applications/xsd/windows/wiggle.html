Create a "wiggle" Window from a Display Window by selecting the "Window→Create Wiggles" menu button. This will bring up the Wiggle Selection Window. Use this window to select the wiggle parameters for creating a Wiggle Display Window from the data in the current Display Window. You can select parameters like the Trace increment and various scaling parameters. The wiggle colors and fill options can also be selected.



The Wiggle Display Window is very much like the normal Display Window except that the data is displayed as Wiggle format instead of color-coded amplitudes. Thus, there is no Contrast or Bias controls nor any "Global Color" toggle capability.

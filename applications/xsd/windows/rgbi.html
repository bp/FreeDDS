The RGBI Window is a composite of three or four Display Window images. The first selected Window image is mapped to the Red color; the second image is mapped to Green; and the third is mapped to Blue. If a fourth Window is selected then it will control the overall intensity level, otherwise the three images will be combined to define the overall intensity level.

The controls are similar to other Display Windows except that the corresponding color panels individually control the image colors:

  • ICont & IBias control the Intensity Level
  • RCont & RBias control the Red Level
  • GCont & GBias control the Green Level
  • BCont & BBias control the Blue Level

When the "Global Colors" toggle is set, the top Intensity color control panel is tied to the top color panel on each Display Window. The second color control panel (Red) is tied to the second color control panel on any Interleave or RGBI Windows. The third (Green) and fourth (Blue) color control panels are tied to any other RGBI Windows.

The pick in xsd is simply a set of 2-Dimensional poly-lines in the 2D plane defined by the first plotted vertical axis and the first plotted horizontal axis. The individual points are called "picks" and each set of picks are called "segments". Every pick segment is contained to the 2D plane from which its first pick is selected. Each segment will be displayed on all the other Display Windows that match the same displayed 2D plane (transposed or not) that has the same axis values for all the other axes. See Saving/Loading to save and restore pick segments.

The current pick/segment editing mode is displayed in the "Edit Mode:" status window. By default each Display Window is setup to begin in the "Picking" mode. Pressing on the left mouse button will add a "pick" to the working "segment"; the middle mouse button will remove the previous "pick"; the right mouse button will close out the picking for the working "segment" and add that "segment" to the list of active segments. The Picks and Segment menus control picking and "segment" editing.

Each Display Window has it's own colormap which can be edited, saved and/or loaded. Additionally, there are 12 colors that can be assigned to different pick segments or cursor. Of these 12 colors one is "white", one is "black" and the other 10 are editable.


  • Pick Segment Colors
    • Edit Segment Color:
      This option allows you to select colors for the pick segment. A color palette Window pops up. Select one of the available 12 colors. Select a segment by pressing the left mouse button to select a segment or press the right mouse button to deselect a segment. Once a segment is selected press the left mouse button again to assign the selected color to the selected pick segment. The color will effect the picks, the lines between the picks, any segment label and/or number. The colors are consistent across all of the Display Windows.
    • Edit Color Palette:
      This option will pop-up the color palette window. Select one of the 10 available colors to change. Click on one of the available colors in the scroll-able list and the press the "Set Color" button to assign that color. Alternatively, just double click on one of the available colors to assign it. Repeat this for any of the other 10 colors on the palette. Press the "Return" button to close the palette window.
  • Cursor Colors
    • Edit Local Cursor Color:
      This option will pop-up a Cursor Color Window. Select the color by specifying the Red, Green and Blue color values (each from 0 to 255) in that order. Any values not given will default to 0. Press the "OK" button to accept the color or "Cancel" to close the window. This color is only applied to the cursor for this Display Window.
    • Edit Global Cursor Color:
      This option Is the same as the "Edit Local Cursor Color" except the color is applied to the cursor in all the Display Windows.
  • Colormaps
    • Load Local Colormap:
      This option will pop-up a Colormap Selection Window. Select a colormap file and press the "OK" button to load it. The colormap will be applied to the current Display Window only.
    • Load Global Colormap:
      This option is similar toe the "Load Local Colormap" except the loaded colormap will be set as the Global Colormap and will be applied to all the current Display Windows and any newly created Display Windows.
    • Use Global Colormap:
      This option will apply the Global Colormap to the current Display Window.
    • Show Colormap Scale:
      This option will pop-up a window displaying the colormap display scale for the current Display Window.
    • Edit Colormap:
      This option will pop-up the Colormap Editor Window. You can "Load" a colormap to edit or the colormap currently loaded. Select which colors (Red, Green and/or Blue) to edit then click a mouse button in the window to alter the color values. You can press a button and drag the cursor to change the values. You can change the "# of points" anywhere from 2 to 200 to specify the number of control points used in the edit window. The colors are interpolated between these points. Press the "Save" button to save the colormap; press the "Apply" button to apply the colormap to the current Display Window. Press the "Flip" button to flip (or reverse) the colormap. Press "Quit" to close the Colormap Editor.

These are the data structures used in teh storage media utility declarations.

Sturctures:

DDS_MEDIA    - Media description (one per GFILE, Unix file descriptor)
DDS_STAT_ID  - Unique file id, based upon stat system call
DDS_AUTO_FMT - Results of automatic format detection on binary stream

Globals:

DDS_MEDIA *dds_media_list; /* media access list (NULL == none) */
char *dds_fmt_name[];      /* Media format (eg. SEGY, DICT, ...) */
char *dds_media_genus[];   /* Media Genus type (eg. REg, FIFO...)*/

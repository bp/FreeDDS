These are the data structures and globals associated with dignostic dump utility declarations.

Sturctures:

DDS_DUMP_DIR - structure per entry in dump list

Globals:

DDS_DUMP_DIR *dds_dump_dir; /* column directory for DDS_DUMP_ITEM. */
                                   /*    Required iff DDS_DUMP_LIST is set. */
                                   /*    Directory shared by all binaries. */
                                   /*    Binaries print data in relevant */
                                   /*    columns or pads them with blanks. */

int dds_dump_max;           /* maximum entry in list or vector. */
                                   /*    If DDS_DUMP_LIST is set, this */
                                   /*       is the number of entries in */
                                   /*       dds_dump_dir and dump_item. */
                                   /*    If DDS_DUMP_DIFF is set, this */
                                   /*       is the maximum number of */
                                   /*       printable vector elements. */

DDS_STACK dds_dump_que;     /* dump "diff", print preamble queue */

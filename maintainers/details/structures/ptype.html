These are the data structures used to define the primary data types.

Sturctures:

DDS_PRIME       - Prime attributes (one for each DDS_PTYPE)
DDS_PRIME_GENUS - type genus "prime" attributes (one for each DDS_TYPE_GENUS)

Globals:

DDS_PTYPE       dds_mutate[];      /* mutate DDS_PTYPE to host "C" type */
DDS_PTYPE       dds_concur[][];    /* concur on type to use between ptypes */
DDS_PRIME      *dds_prime;         /* Primary data types */
DDS_PRIME_GENUS dds_prime_genus[]; /* Genus of ptypes (eg. FLOATS, INTEGERS...) */

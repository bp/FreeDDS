These are the data structures used to define the binaries for each dataset.

Structures:

DDS_SEAM       - Expression sequence for read, write, or map evaluation
DDS_OBJECT_REF - Symbol table object reference
DDS_MEMBER_REF - Symbol table member reference
DDS_BIN_USP    - Binary descriptor, usp standard
DDS_BIN_SEGY   - Binary descriptor, segy standard
DDS_BIN_GEN    - Binary descriptor, generic standard: zip-compressed table
DDS_AXIS       - Axis description for hyper-cubes
DDS_CUBE       - Hyper-cube description
DDS_BIN        - Binary descriptor (one per open binary)
DDS_MAP        - "map" descriptor, unique combination (see DDS_ROOT.map)

Globals:

DDS_BIN *dds_compile;    /* Compile binary descriptor.  The parser and scanner
                            append symbols to the tables referenced by this. */

DDS_VALUE *dds_map_value;/* Compiler handle to "in" struct. Non-NULL iff 
                            compiling maps & input buffer type is struct/union */

int dds_map_samples;     /* Switch for cddx_outhdr to turn off mapping Samples */

unsigned dds_bin_max;    /* Maximum number of DDS_BIN pointers */
DDS_BIN **dds_bin;       /* Binary descriptor table (bin_tag); bin[bin_max] */
DDS_MAP **dds_map;       /* Map descriptor table (out/in bin_tag); map[bin_max][bin_max] */

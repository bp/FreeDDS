Svn:

This is a dummy subversion version definition for "SVN_ID" in case the makefile structure doesn't create one. Normally, the DDS application make files will generate a file named "svn_version.c" in the object subdirectory everytime the application is built. This way the linked code will always have the latest version number of the application available for listing in the both the printout files and any generated output dictionaries.

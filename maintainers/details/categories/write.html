Write:

These are only called through cdds_write. 

Files:
    f_xxx.c        - Fortran API Routines
                        f_writem.c  f_write.c
    c_xxx.c        - "C" API Routines
                        c_writem.c c_write.c 
    writeuspl.c   - Write usp line header binary data
    writeuspt.c   - Write and edit usp trace binary data
    writesegyc.c  - Write segy card header binary data
    writesegyb.c  - Write segy binary header binary data
    writesegyt.c  - Write and edit segy trace binary data
    writegent.c   - Write and edit generic trace binary data
    writegen.c    - Write generic binary data on a byte-basis
    writeslicet.c - Write and edit sliced trace binary data
    writeslice.c  - Write sliced binary data on a byte-basis
    writecheck.c  - Check results of write binary data
                    (residual bytes, EOF...)

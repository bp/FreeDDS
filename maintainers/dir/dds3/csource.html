This is the C Source code naming conventions for the DDS3 outines. 

    cdds.h      C API include file
    c_*.c       cdds_*: C API
    cx_*.c      cddx_*: C API convenience routines

    f_*.c       fdds_*: Fortran API (wrapper for C version)
    fx_*.c      fddx_*: Fortran API convenience routines (wrapper for C)
    fdds.h      Fortran API include file (arcitecture specific filename)
                  Derived from running "makefdds" on f_com.h
    f_com.h     Fortran include, common to all dialects

    makecon2.c  used to generate intrinsic include file, dds_con2.h
    makedict.c  used to compile intrinsic dictionaries (*.dds) into 
                include files (*.h) to be included in the libraries
    makefdds.c  used with f_com.h & f90_com.h to generate
                  fdds.h & f90dds.h

    *.c         ddsXXX: Other routines called internally

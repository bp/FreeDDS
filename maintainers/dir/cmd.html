This directory contains all the DDS application codes, each one under it's own unique directory by application name.

${DDSROOT}/src/cmd: Subdirectory containing a directory for each of the applications. Each application directory will contain:

  • Local scripts:
    • BuildAll - Build all applications for all remote hosts
    • BuildHost - Build all applications for current host
  • Local files under each application directory:
    • Makefile - make file to build & install the startup script, and executable and, if it exists, the man page
    • architecture-dependent subdirecties containing the working build files
    • source code
    • man page

This directory contains the fftw routines. These are only fall-back routines for when we don't have a good alternative on a specify platform. The make files have been modified to use the DDS make file structure instead of what comes with fftw.

${DDSROOT}/src/lib/fftw: Generates libfftw.a, librfftw.a, fftw.h and rfftw.h (fftw routines for when there is no other host-dependent fft library) for building DDS applications.

  • Local files & directories:
    • Makefile - master fftw make file
  • Libraries that gets installed under ${DDSROOT}/lib:
    • libfftw.a - fftw complex-complex API routines available to be used by fft wrappers
    • librfftw.a - rfftw real-complex API routines available to be used by fft wrappers
  • Include files that gets installed under ${DDSROOT}/include:
    • fftw.h - fftw include file available to be used by fft wrappers
    • rfftw.h - rfftw include file available to be used by fft wrappers

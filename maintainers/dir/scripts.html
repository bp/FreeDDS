This directory contains several user application scripts.

${DDSROOT}/src/scripts: Installs various DDS utility scripts into ${DDS_DIST}/bin.

  • Local files:
    • Makefile - make file to install scripts and any associated man pages
    • makedict - script to assis creating a super Dictionary of sliced datasets
    • asciidump - script to dump Samples from aa DDS file to an ascii format
    • mbs-list - script to list all the available executable binaries for a specific application
    • mbs-which - script to show which binary would be executed on the current host when the application script is run
    • mbs-show - script to show any old application executables; the date code can be used by the scripts with "old=" to execute any of these available old binaries
    • dds_axis - script to show the axis information from a DDS Dictionary
    • dds_what -script to show what applications have been run to generate a DDS Dictionary
    • dds_when -script to show when applications were run on a DDS Dictionary
    • dds_where -script to show where the applications were run on a DDS Dictionary
    • dds_who -script to show who ran applications on a DDS Dictionary
    • dds_www -script to show who, what, when & where from a DDS Dictionary

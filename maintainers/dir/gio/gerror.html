gerror, geof, gclearerr, gfileno - status inquiries for GIO stream

SYNOPSIS

#include "gio.h"
int gerror(GFILE *stream);
int geof(GFILE *stream);
int gclearerr(GFILE *stream);
int gfileno(GFILE *stream);

DESCRIPTION

See standard man pages for ferror(3), feof(3), clearerr(3), fileno(3). See gio, for an overview of GIO and comparision to stdio.

FILES

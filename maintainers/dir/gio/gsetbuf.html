gsetbuf, gsetbuffer, gsetvbuf - assign buffering to a GIO stream

SYNOPSIS

#include "gio.h"
int gsetvbuf(GFILE *stream, void *buf, int mode, size_t size);

DESCRIPTION

See standard man pages for setbuf(3), setbuffer(3), setvbuf(3). See gio for an overview of GIO and comparision to stdio.

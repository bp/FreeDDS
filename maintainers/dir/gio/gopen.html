gopen, greopen, gdopen - open a GIO stream

SYNOPSIS

#include "gio.h"
GFILE gopen(const char *filename, const char *type);
GFILE greopen(const char *filename, const char *type, GFILE *stream);
GFILE gdopen(int filedes, const char *type);

DESCRIPTION

See standard man pages for fopen(3), freopen(3), fdopen(3).

See gio(3), for an overview of GIO and comparision to stdio.

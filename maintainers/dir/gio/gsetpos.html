gseek, gtell - reposition a file pointer in a GIO stream

SYNOPSIS

#include "gio.h"
int gsetpos(GFILE *stream, const gpos_t *offset);
int ggetpos(GFILE *stream, gpos_t *offset);

DESCRIPTION

See standard man pages for fsetpos(3), fgetpos(3). See gio for an overview of GIO and comparision to stdio.

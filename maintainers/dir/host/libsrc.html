These are the C Source files that make up the libchost.a and libfhost.a libraries
  • ascii_ebcdic.c - Routines for conversion between ascii and ebcdic character. This is included in both libchost.a and libfhost.a.

  • int8.c - Routines for handling 8-bytes integers when it is not available and thus "HOST_FAKE8" is set to fake the use of 8-byte integer. This is included in both libchost.a and libfhost.a.

  • getargc.c - "C" routines for handling command line arguments and environmental variables. This is included in libchost.a.

  • getargf.c - Fortran routines for handling commnand line arguments and environmental variables. This is included in libfhost.a.

  • lnblnk.c - Fortran routine for "lnblnk" when not available on some systems. This is the routine for returning the length of a character string, ignoring any trailing blanks. This is included in libfhost.a.

These are some programs to assist other miscellaneous operations
  • digestWiki.c - This program is used when an application program gets installed. It searched the wiki for any information on the program to append at the end of the manpage that gets installed.

  • makehost.c - This program is used to generates (along with chost_com.h) chost.h. This allows most of the host- & compile-dependent characteristics to be captured automatically instead of the older error-prone manual setup.

  • tilde.c - This program was used to generates tables used in ascii_ebcdic.c

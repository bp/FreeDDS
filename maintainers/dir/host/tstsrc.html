Host Utilities:

There are several host utilities to assist in configuring DDS for a specific host machine when and if need be.

  • ctest1.c - Prints "Hello World"
  • ctest2.c - Prints out various compiler variable sizes
  • cdefine.c - Prints out various define types
  • testhost.c - Tests for data type consistencies
  • ctestargcv.c - Tests setargcv linkage
  • ctypes.c - Prints out various compiler data type information
  • Ftest1.F/ftest.h/lsize.c - test the Fortran linkage & command line arguments

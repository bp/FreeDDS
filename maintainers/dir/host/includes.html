• chost.h - host-dependent include file used to build other DDS routines. This is generated by makehost.c and gets installed under ${DDSROOT}/include/${TARCH}.

  • chost_com.h - common include file that gets appended to chost.h.

  • ascii_ebcdic.h - include file for ASCII/EBCDIC conversion tables. This gets installed under ${DDSROOT}/include.

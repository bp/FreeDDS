This directory contains the documentation for DDS in the form of web pages. The documentation for each of the DDS applications are in the form of man pages which reside under "${DDS_DIST}/man/mbs/man1" directory. "PHP" is used for the search engine and for displaying application man pages. Also, there is a script ("${DDS_DIST}/doc/html/applications/buildmanlist") which is used to keep a regular update of the applications pages.
